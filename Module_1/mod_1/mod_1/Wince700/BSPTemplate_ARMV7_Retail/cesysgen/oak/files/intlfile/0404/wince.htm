    啟動程式

      若要開啟非位於桌面上的程式，請選取 [開始] 功能表，選取 [程式集] ，並選取程式名稱。


      使用線上即時說明
      • 如需關於程式或對話方塊的資訊，請選取程式或對話方塊中的 [?]按鈕。
      • 若要檢視工具列按鈕的名稱，請使用手寫筆點選並按住按鈕。若要避免執行該按鈕指令，請繼續按住按鈕，並將手寫筆移到按鈕以外的地方放開。
      • 若要使用滑鼠檢視工具列按鈕的名稱，請將滑鼠游標移到按鈕上方。

      開啟範圍功能表
      • 使用手寫筆時，點選 ALT 鍵，再點選項目以開啟範圍功能表。
      • 使用滑鼠時，以滑鼠右鍵按一下項目，即可開啟範圍功能表。

                            關於 Pocket Internet Explorer

                              使用 Pocket Internet Explorer，您可於個人裝置中檢視 Internet 或 Intranet 網頁。您必須使用數據機或乙太網路卡來與 ISP 或網路連線。

                                  連線到 Internet

                                    您必須先設定 ISP 或網站的遠端連線，才能檢視網站。若您尚未設定 ISP 或網路的連線，請查閱「遠端連線」。

                                    1. 選取 [開始] > [程式集] > [通訊] > [遠端網路]。 雙點或按兩下您所要使用的連線。
                                    2. 輸入所有必要的使用者資訊。 若要調整撥號方式，請選取 [撥號內容] 。
                                    3. 選取 [連線] 。

                                    附註 若無法連線，可能必須使用代理伺服器來存取 Internet。

                                      請參閱
                                      設定代理伺服器

                                      設定 Internet 自動連線

                                        若設定 Internet 自動連線，當 Pocket Internet Explorer 需要存取 Internet 或公司網路時，即會自動撥號來執行遠端連線。

                                        1. 選取 [檢視] > [選項] > [自動撥號] 索引標籤。
                                        2. 選取 [使用自動撥號] 。
                                        3. 指定要使用的遠端連線。
                                        4. 若想要在一定的閒置時間 (個人裝置未於使用狀態的時間) 過後，即自動離線，請選取 [自動離線] 。
                                        5. 在 [怠機達] 中，輸入個人裝置從 Internet 離線的閒置時間長度。
                                        請參閱
                                        連線到 Internet

                                        設定代理伺服器

                                          代理伺服器通常在透過本地網路連線到 Internet 時所使用，例如公司網路，以增加安全性。

                                          1. 選取 [檢視] > [選項] > [代理伺服器] 索引標籤。
                                          2. 選取 [使用代理伺服器]。
                                          3. 輸入代理伺服器的位址和通訊埠。如需更多資訊，請聯繫您的網路管理員。
                                          4. 若要本地端的位址略過代理伺服器，例如公司內部網站頁面，請選取 [近端網址不使用 Proxy]。
                                          請參閱
                                          連線到 Internet
                                          選取安全性選項

                                          啟動 Cookie

                                            Cookie 檔案含有關於您的識別和喜好設定等資訊，所以網站可據此來量製適合個人需求的資訊。網站會傳送該檔案，並將檔案儲存於個人行動裝置中。

                                            1. 選取 [檢視] > [選項] > [進階] 索引標籤。
                                            2. 選取 [啟動 Cookie] 。

                                                    儲存目前的網頁

                                                      當您從 Internet 或 Intranet 離線時，可儲存網頁並於稍後檢視這些網頁，而不需使用快取記憶體。

                                                      1. 選取 [檔案] > [另存新檔] 。
                                                      2. 指定資料夾的位置並指定檔名。

                                                      注意

                                                      • 影像將不會顯示出來，同時連結也會失效。
                                                      • 含有圖文框的網頁將不會顯示或僅呈現圖文框的結構。
                                                      • 若要儲存個別圖文框，請按住 ALT 並點選或按一下該圖文框。選取範圍功能表上的 [另存新檔] 或 [另存圖片]。某些圖文框經過特殊設計，您無法儲存這些圖文框。當您從 Windows 的「檔案總管」以儲存的檔案方式開啟這些圖文框時，它們並不會顯示。

                                                      請參閱
                                                      離線時檢視快取網頁

                                                      離線時檢視快取網頁

                                                        當您在 Internet 上檢視網頁時，這些網頁會快取至「我的電腦」中的「Temporary Internet Files」資料夾中。在快取網頁間瀏覽會比回到 Internet 網站的速度要來得快。

                                                        1. 離線前，請選取 [檔案] > [離線工作] 。
                                                        2. 當您已離線時，請選取 [移至] > [記錄] 。
                                                        3. 選取您要檢視的網頁。

                                                        注意

                                                        • 連結將會失效。
                                                        • 當檢視使用圖文框的快取網頁時，只有每個圖文框集的第一個圖文框會顯示出來。某些圖文框由於原始設計而無法快取。當您以快取檔案方式開啟時，這些圖文框仍將無法顯示。
                                                        請參閱
                                                        調整快取記憶體的設定

                                                        調整快取記憶體的設定

                                                          預設情況下，Pocket Internet Explorer 的快取記憶體比率佔個人裝置總記憶體儲存量的百分之十。若有記憶體儲存方面的問題，可試著清除 Pocket Internet Explorer 的快取記憶體。

                                                          1. 選取 [檢視] > [設定] > [進階] 索引標籤。
                                                          2. 若要使用快取作業 (暫時儲存區)，請選取 [啟動快取] 。
                                                            請注意：快取作業會消耗記憶體。
                                                          3. 移動 [快取記憶體大小] 滑動桿來增加或減少快取記憶體佔儲存記憶體的比例。
                                                          4. 若要在您關閉 Pocket Internet Explorer 時，清空該資料夾，請選取 [結束時清空快取記憶體] 。
                                                          5. 若要立即清空資料夾，請選取 [立即清空快取空間] 。

                                                            關於安全性

                                                              Pocket Internet Explorer 支援安全性網站中使用的安全性通訊協定。當存取安全性網站時，您可安全地傳送個人資訊。若要查看是否為安全性網站，請於狀態列中的 Pocket Internet Explorer 按鈕下中檢查鎖定圖示。

                                                                請參閱
                                                                選取安全性選項

                                                                選取安全性選項
                                                                1. 選取 [檢視] > [選項] > [進階] 索引標籤。
                                                                2. 選取 [安全性設定] 。
                                                                3. 選取您要的安全性通訊協定。
                                                                  • PCT 是私人通訊科技 (Private Communications Technology)，為 Microsoft 所發展出來的專屬通訊協定，其安全層次較 SSL2 為高。Windows Embedded Compact OS 不支援 PCT。
                                                                  • SSL2 為 [Secured Sockets Layer Level 2]，是一種安全傳輸的標準通訊協定。所有的安全性 Internet 網站均支援此通訊協定。
                                                                  • SSL3 為 [Secured Sockets Layer Level 3]，為一專屬的通訊協定，其安全層次較 SSL2 為高。某些 Internet 網站不支援此通訊協定。
                                                                請參閱
                                                                連線到 Internet
                                                                啟動 Cookie

                                                                非安全性網站

                                                                  您正要進入的網站未具有憑證，也不是個安全性網站。依據您已知的個人裝置及此網站資訊，您必須決定是否仍要繼續進入此網站。若不確信是否要進入此網站，請選取 [否] 。

                                                                        保護重要資訊

                                                                          您可能會將重要資訊保存於個人裝置上，例如：名稱、重要連絡人的電話號碼、信用卡號碼及機密文件。下列為保護重要資訊的方法。

                                                                            • 使用裝置鎖定密碼。
                                                                              您可以設定密碼保護您的裝置，任何人要使用您的裝置前都必須提供密碼。選取 [開始] > [設定] > [控制台]。 開啟 [密碼] 圖示。

                                                                              重要
                                                                              選擇容易記住的密碼。若忘記密碼，您必須執行完整重置，才能再使用您的裝置。完整重置會消除您所有的資料。

                                                                            • 備份您的資料。
                                                                              您可以建立並維護備份檔案，將裝置的程式資料放到備份中。假設您因為意外移除主電池與備用電池，而導致資料消失，就可以從備份檔回存資料。
