    Iniciando programas

      Para abrir um programa que não está na área de trabalho, selecione o menu Iniciar, selecione Programas e, em seguida, selecione o nome do programa.


      Utilizando a Ajuda do contexto
      • Para obter informações sobre um programa ou sobre uma caixa de diálogo, selecione o botão ? enquanto estiver usando o programa ou a caixa de diálogo.
      • Para ver o nome de um botão da barra de ferramentas, toque e segure com a caneta no botão. Para evitar que o comando do botão seja executado, continue mantendo o botão pressionado enquanto arrasta a caneta para fora do botão.
      • Para ver o nome de um botão da barra de ferramentas usando o mouse, coloque o ponteiro do mouse sobre o botão.

      Abrindo um menu de contexto
      • Se estiver usando uma caneta, toque a tecla ALT e, em seguida, toque no item desejado para abrir um menu de contexto.
      • Se estiver usando um mouse, clique com o botão direito no item desejado para abrir um menu de contexto.

              Alternando entre menus e botões da barra de ferramentas

                Use o controle deslizante para controlar a exibição de menus e da barra de ferramentas.

                • Para alternar rapidamente entre menus e botões da barra de ferramentas, selecione o controle deslizante.
                • Arraste o controle deslizante à esquerda ou à direita para exibir menus, botões da barra de ferramentas ou uma combinação de ambos.
                • Para verificar menus e botões da barra de ferramentas, arraste o controle deslizante abaixo dos menus.


                Movendo um arquivo para uma placa de memória

                1. Abra o arquivo a ser movido.
                2. Selecione Arquivo > Salvar como.
                3. Salve o arquivo na pasta nomeada para a placa de memória.

                      Avançando e voltando
                      • Para ir para a pasta anterior, selecione Iniciar > Voltar.
                      • Para ir para a próxima pasta (que foi recém-verificada), selecione Iniciar > Avançar.

                        Criando uma nova pasta
                        1. Se necessário, abra uma pasta na qual você quer colocar a nova pasta.
                        2. Selecione Arquivo > Nova pasta.

                        Adicionando uma pasta à lista Favoritos
                        1. Abra a pasta.
                        2. Selecione Favoritos > Adicionar a Favoritos.
                        3. Na caixa Nome, digite o nome do atalho.
                        Dicas

                        • Para enviar um arquivo, selecione-o, alinhe as portas de infravermelho e selecione Arquivo > Enviar para > Destinatário de infravermelho.
                        • Para receber um arquivo, alinhe as portas de infravermelho e selecione Arquivo > Receber.

                        Criando atalhos da área de trabalho
                        1. Exiba o arquivo ou a página da Web a ser criado um atalho.
                        2. Selecione Arquivo > Enviar para.
                        3. Selecione Desktop como atalho.

                            Sobre o Pocket Internet Explorer

                              Com o Pocket Internet Explorer, é possível exibir páginas da Web da Internet ou de uma intranet no dispositivo. Será necessário usar um modem ou uma placa Ethernet para conectar-se ao provedor de serviços da Internet (ISP) ou à rede.

                                  Conectando-se à Internet

                                    Antes que seja possível exibir páginas da Web, você deve configurar uma conexão remota ao ISP ou à rede. Se você não configurou uma conexão ao ISP ou à rede, consulte Conexões remotas.

                                    1. Selecione Iniciar > Programas > Comunicação > Rede remota. Toque ou clique duas vezes na conexão que deseja usar.
                                    2. Insira qualquer informação do usuário necessária. Para ajustar a maneira como o número deve ser discado, selecione Propriedades de discagem.
                                    3. Selecione Conectar.

                                    Observação Se houver problemas na conexão, convém acessar a Internet usando um servidor Proxy.

                                      Consulte também
                                      Configurando um servidor proxy

                                      Configurando uma conexão automática à Internet

                                        Se você configurou uma conexão automática à Internet, a conexão remota escolhida será discada automaticamente quando o Pocket Internet Explorer precisar acessar a Internet ou a rede associada.

                                        1. Selecione Exibir > Opções > guia Discagem automática.
                                        2. Selecione Usar discagem automática.
                                        3. Especifique a conexão remota a ser utilizada.
                                        4. Se você quiser desconectar-se automaticamente da Internet depois de um certo tempo de inatividade (tempo em que o dispositivo não foi utilizado), selecione Desconexão automática.
                                        5. Em Desconectar se inativo por, insira quanto tempo o dispositivo deve estar inativo antes de desconectar-se da Internet.
                                        Consulte também
                                        Conectando-se à Internet

                                        Configurando um servidor proxy

                                          Os servidores proxy são normalmente utilizados quando a conexão à Internet é feita por meio de uma rede local, como uma rede associada, para aumentar a segurança.

                                          1. Selecione Exibir > Opções > guia Servidor proxy.
                                          2. Selecione Usar servidor proxy.
                                          3. Insira o endereço e a porta do servidor proxy. Para obter mais informações, consulte o administrador da rede.
                                          4. Para ignorar o servidor proxy para endereços locais, como páginas da intranet associada, selecione Ignorar Proxy para endereços locais.

                                                    Salvando uma cópia da página atual

                                                      É possível salvar páginas da Web e exibi-las posteriormente quando você estiver desconectado da Internet ou de uma intranet sem usar a memória cache.

                                                      1. Selecione Arquivo > Salvar como.
                                                      2. Especifique um local de pasta e nomeie o arquivo.

                                                      Observações

                                                      • Imagens não serão exibidas e vínculos podem ser desativados.
                                                      • Páginas da Web contendo quadros não serão exibidas ou somente a estrutura do quadro será exibida.
                                                      • Para salvar um único quadro, pressione ALT e toque ou clique no quadro. Selecione Salvar como ou Salvar figura como no menu Contexto. Alguns quadros são projetados de forma que não possam ser salvos. Esses quadros não serão exibidos quando você os abrir como arquivos salvos no Windows Explorer.

                                                      Consulte também
                                                      Exibindo uma página em cache enquanto desconecta-se

                                                      Exibindo uma página em cache enquanto desconecta-se

                                                        À medida que você visualiza páginas da Internet, elas são armazenadas em cache na pasta Temporary Internet Files em Meu computador. A procura entre páginas em cache é mais rápida que retornar a seus sites na Internet.

                                                        1. Antes de desconectar, selecione Arquivo > Trabalhar off-line.
                                                        2. Uma vez desconectado, selecione Iniciar > Histórico.
                                                        3. Selecione a página desejada.

                                                        Observações

                                                        • Os vínculos serão desativados.
                                                        • Ao exibir páginas da Web em cache com quadros, somente o primeiro quadro de cada conjunto será exibido. Alguns quadros são desenvolvidos de modo que não ficam em cache. Esses quadros não serão exibidos se não forem abertos como arquivos em cache.

                                                        Ajustando configurações para memória cache

                                                          Como padrão, a cache do Pocket Internet Explorer ocupa 10 por cento do armazenamento em memória total do dispositivo. Se houver problemas com o armazenamento em memória, tente limpar a cache do Pocket Internet Explorer.

                                                          1. Selecione Exibir > Opções > guia Avançada.
                                                          2. Se você quiser permitir o armazenamento em cache (memória temporária), selecione Ativar cache.
                                                            Observe que o armazenamento em cache irá drenar memória.
                                                          3. Mova o controle deslizante Tamanho de cache para aumentar ou diminuir o armazenamento destinado à cache.
                                                          4. Se você quiser que a pasta seja esvaziada quando você fechar o Pocket Internet Explorer, selecione Esvaziar cache ao sair.
                                                          5. Para esvaziar a pasta imediatamente, selecione Esvaziar cache agora.

                                                            Sobre segurança

                                                              O Pocket Internet Explorer suporta os protocolos de segurança utilizados por sites seguros. Ao acessar um site seguro, é possível enviar informações pessoais com segurança. Para determinar se um site da Web é seguro, verifique se há um ícone de proteção no botão do Pocket Internet Explorer na barra de status.

                                                                Consulte também
                                                                Selecionando opções de segurança

                                                                Selecionando opções de segurança
                                                                1. Selecione Exibir > Opções > guia Avançada.
                                                                2. Selecione Configurações de segurança.
                                                                3. Selecione os protocolos de segurança desejados.
                                                                  • PCT significa Private Communications Technology, um protocolo patenteado desenvolvido pela Microsoft que é significativamente mais seguro que SSL2. O sistema operacional Windows Embedded Compact não oferece suporte a PCT.
                                                                  • SSL2 significa Secured Sockets Layer Level 2, um protocolo padrão para transmissões seguras. Todos os sites seguros da Internet suportam esse protocolo.
                                                                  • SSL3 significa Secured Sockets Layer Level 3, um protocolo patenteado que pretende ser mais seguro que o SSL2. Alguns sites da Internet não suportam esse protocolo.

                                                                        Protegendo informações importantes

                                                                          É possível manter dados valiosos no dispositivo, como nomes e números de telefone de contatos importantes, números de cartão de crédito e documentos confidenciais. Aqui estão algumas maneira de proteger as informações importantes.

                                                                            • Use uma senha para bloqueio de dispositivo.
                                                                              Você pode definir uma senha a ser digitada para que você ou qualquer outro usuário tenha acesso ao dispositivo. Selecione Iniciar > Configurações > Painel de Controle. Abra o ícone Senha.

                                                                              IMPORTANTE
                                                                              Escolha uma senha de fácil memorização. Se esquecê-la, será necessário executar uma redefinição completa para usar o dispositivo novamente. A redefinição completa irá apagar todos os seus dados.

                                                                            • Faça backup dos dados.
                                                                              Você pode criar e manter um arquivo de backup com os dados de programas de seu dispositivo. Se, por exemplo, você remover acidentalmente as baterias principais e de reserva e perder seus dados, poderá restaurá-los a partir do arquivo de backup.

                                                                                                                              Caixa de diálogo Cor

                                                                                                                                Cores básicas
                                                                                                                                Mostra as cores básicas disponíveis. Para definir uma cor personalizada, selecione a cor básica mais próxima a ela e, em seguida, selecione Definir.

                                                                                                                                  Cores personalizadas
                                                                                                                                  Exibe todas as cores personalizadas que você definiu.
                                                                                                                                  Para alterar as configurações de uma cor personalizada, clique em uma caixa de cor personalizada e, em seguida, selecione uma cor diferente.
                                                                                                                                  Para definir uma nova cor personalizada, clique em uma caixa de cor personalizada vazia, selecione uma cor diferente e, em seguida, selecione Adicionar a Cores personalizadas.

                                                                                                                                    Definir
                                                                                                                                    Define uma cor personalizada. Se não estiver disponível, você já está exibindo cores personalizadas.
                                                                                                                                    Para definir uma cor personalizada, selecione a matriz de cor ou selecione a cor usando as definições de matiz, de saturação e de luminosidade ou as definições vermelho, verde e azul.

                                                                                                                                      Adicionar a Cores personalizadas
                                                                                                                                      Adiciona a cor personalizada que você definiu.

                                                                                                                                        Cor/Sólido
                                                                                                                                        À esquerda, essa caixa exibe a quantidade de branco e de preto na cor atual. À direita, exibe 100% da cor sem branco e sem preto. Ajuste usando o controle deslizante à direita.
                                                                                                                                        Para selecionar 100% de cor sem branco ou sem preto, digite ALT+O.
                                                                                                                                        Para alterar a cor, altere os valores descritos abaixo. Ou faça experiências selecionando e arrastando a matriz de cor.

                                                                                                                                          Matiz
                                                                                                                                          O valor de uma roda de cor. Vermelho = 0, Amarelo = 60, Verde = 120, Ciano = 180, Magenta = 200 e Azul = 240.

                                                                                                                                            Sat
                                                                                                                                            Nível de saturação. A quantidade de cor em uma matiz específica, até 240.

                                                                                                                                              Lum
                                                                                                                                              Luminosidade ou brilho. À medida que você altera a luminosidade, os valores de vermelho, de verde e de azul alteram-se.

                                                                                                                                                Vermelho
                                                                                                                                                A quantidade de vermelho na cor selecionada.

                                                                                                                                                  Verde
                                                                                                                                                  A quantidade de verde na cor selecionada.

                                                                                                                                                    Azul
                                                                                                                                                    A quantidade de azul na cor selecionada.

                                                                                                                                                                                      Configurando uma senha para um arquivo
                                                                                                                                                                                      1. Selecione Arquibo > Senha.
                                                                                                                                                                                      2. Digite e verifique a senha.
                                                                                                                                                                                      3. Você deve salvar o arquivo (Arquivo > Salvar) para reter a senha.

                                                                                                                                                                                      Observação Não é possível sincronizar arquivos protegidos por senha. Você deve remover a proteção por senha no programa apropriado no PC ou no dispositivo e, em seguida, fazer a sincronização. Uma vez sincronizado, você pode redefinir a proteção por senha.
