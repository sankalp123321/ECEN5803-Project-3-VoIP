憑證概觀

憑證是種數位擔保，包含提示該擔保之裝置的相關資訊。例如，Web 伺服器這種裝置可以擁有包含本身相關資訊的憑證。有效的憑證是由憑證機構 (CA) 所簽發，並包含憑證簽發者、憑證之公開金鑰與憑證到期日等資訊。

連線到 Internet 時，信任您所瀏覽之網站的憑證是非常重要的。瀏覽未受信任的網站或未擁有適當憑證的網站時，您裝置所儲存的資料可能會有外洩的風險。如果您不確定，可以從您的裝置檢視該網站的憑證，並決定是否繼續瀏覽該網站。

您裝置的安全性與隱私權功能可幫助您保護您的隱私，並確保您的個人資訊是安全的。這些隱私權功能可讓您保護您的個人識別資訊 (PII)。這些功能可幫助您控制是否要讓您所瀏覽的網站存取您的個人識別資訊。藉由使用您裝置上的隱私權控制功能，您也可以指定隱私權設定，以決定瀏覽網站時要共用哪些資訊，以及允許網站將哪些資訊儲存到您的裝置。

用戶端憑證

用戶端憑證是數位聲明，可證實使用者或裝置 (例如個人電腦) 的身份。用戶端憑證簽發的目的是建立識別、提供驗證等級與保護各種裝置交換資訊時的安全性。某些網站要求您使用用戶端憑證來識別您的身份。當您傳送用戶端憑證到另一部電腦時，您會使用允許您以安全方式與該裝置共用資料的公開與私密金鑰。公開與私密金鑰也能幫助您防止未經授權的使用者存取您的裝置與裝置上的資訊。

下列清單顯示憑證中包含的資訊：

  • 對象的公開金鑰值與識別資訊
  • 簽發者的識別資訊
  • 有效期間
  • 簽發者的數位簽章

對象是收到憑證的實體；憑證的簽發者是憑證機構。

個人與根憑證

您可以在您的裝置上儲存兩種類型的憑證：個人憑證是用以建立您的識別；根憑證則用以建立 Web 伺服器的識別。例如，當您透過 Internet 傳送個人資訊時，會將您的個人憑證交由您瀏覽的網站驗證，而根憑證則幫助您在瀏覽網站時保護您的個人資訊。

隱私權概觀

藉由使用您裝置上的隱私權功能，可以保護您的個人識別資訊 (PII)。隱私權功能可幫助您控制是否要讓您所瀏覽的網站存取您的個人識別資訊 (PII)。這些功能可讓您指定隱私權設定，以決定是否要允許您所瀏覽的網站將資訊 (例如 cookie) 儲存到您的個人電腦。

Cookie

每次您瀏覽網站時，該網站可能會儲存一個小檔案 (稱為 cookie) 到您的個人電腦。網站建立 cookie 以儲存資訊，例如您瀏覽該網站時的喜好設定。某些 cookie 包含您在某網站中瀏覽的網頁記錄，這些記錄可幫助網站根據您的瀏覽習慣自訂其網頁。

Cookie 也可以儲存個人識別資訊，例如您的姓名、電子郵件地址、住家或公司地址或電話號碼。不過，網站只能存取您所提供的個人識別資訊。

下列清單顯示您裝置上的隱私權功能：

  • 隱私權設定，指定您的電腦將如何處理 cookie
  • 隱私權警告，在您嘗試存取不符合您指定之隱私權設定的網站時通知您
  • 可檢視網站的 P3P 隱私權原則

您可以使用裝置上的隱私權功能來限制 cookie 的使用。不過，因為許多應用程式依存於 cookie 中所包含的資訊，停用所有 cookie 並不是最好的方式。

第一方與第三方內容

第一方內容包含主機網域或您正在瀏覽之網站的相關資訊。第三方內容則來自任何其他網域。例如，當您瀏覽包含橫幅廣告的網站時，您正在瀏覽的網站會被認為是第一方內容，而橫幅廣告則會被認為是第三方內容。

許多商業網頁都使用第一方與第三方 cookie 之組合來收集並傳送與這些網頁相關的資料。您裝置上的隱私權功能可以分辨第一方與第三方內容，並與第一方與第三方建立不同的關係。事實上，對第三方內容的預設隱私權設定比第一方的嚴格。您 Windows Embedded Compact 裝置上的隱私權設定允許您停用第三方 cookie 而允許第一方 cookie，或提示您允許或拒絕這些 cookie。

安全性概觀

您裝置上的安全性功能可幫助您防止未經授權的使用者存取您裝置上的資訊，例如您在 Internet 上購物時所輸入的信用卡資訊。安全性功能也能保護您的個人電腦或無線裝置，避免受到不安全軟體的影響。

使用 Hypertext Transfer Protocol over Secure Socket Layer (HTTPS) 通訊協定瀏覽網站時，瀏覽的安全性就會提高。HTTPS 是安全的 Web 通訊協定，在這種連線上傳送的資料都會經過加密。第三方無法攔截並解讀使用 HTTPS 加密的資料。

安全性警告

發生下列情況時，會出現安全性警告：

  • 您正在連線的 Web 伺服器包含無效的伺服器憑證
  • 您正在連線的 Web 伺服器包含混合安全性 (安全與不安全的資料)
  • 您將離開安全連線
