    Informationen zu WordPad

      Sie können in WordPad Dokumente und Vorlagen unter Verwendung von Schaltflächen und Menübefehlen erstellen und bearbeiten, die Ihnen durch die Desktop-Version von Microsoft Word vertraut sind. Sie können mit Dateien entweder in der Normal- oder Gliederungsansicht arbeiten.

        WordPad-Dokumente werden normalerweise als .pwd-Dateien gespeichert; Sie können Dokumente jedoch auch mit anderen Dateiformaten, wie z.B. .rtf oder .doc speichern.

            Markieren von Text
              Um Text zu markieren führen Sie die folgenden Schritte aus
              Beliebige Menge an Text Ziehen Sie den Stift oder den Mauszeiger über den Text.
              Ein Wort Doppeltippen oder doppelklicken Sie auf das Wort.
              Ein einzelner Absatz Tippen oder klicken Sie dreimal innerhalb des Absatzes.
              Gesamtes Dokument Klicken Sie auf Bearbeiten > Alles markieren.
              Ein Bild Tippen oder klicken Sie auf das Bild.


              Suchen von Text
              1. Um nach Text in einem bestimmten Teil des Dokuments zu suchen, markieren Sie den Teil.
                Wenn Sie den Teil nicht markieren, wird das gesamte Dokument durchsucht.
              2. Klicken Sie auf Bearbeiten > Suchen.
              3. Geben Sie den zu suchenden Text ein.
                Geben Sie für einen Absatz ^p und für einen Tabstopp ^t ein.
              4. Wählen Sie die gewünschten Suchoptionen aus.
              5. Klicken Sie auf Weitersuchen.
              6. Um die Suche fortzusetzen, klicken Sie auf Bearbeiten > Weitersuchen.

              Ersetzen von Text
              1. Um Text in einem Teil des Dokuments zu ersetzen, markieren Sie den entsprechenden Teil.
                Wenn Sie den Teil nicht markieren, wird das gesamte Dokument durchsucht.
              2. Klicken Sie auf Bearbeiten > Ersetzen.
              3. Geben Sie den Text ein, den Sie suchen möchten, und den Text, durch den Sie den gesuchten Text ersetzen möchten.
                Geben Sie für einen Absatz ^p und für einen Tabstopp ^t ein.
              4. Wählen Sie die gewünschten Optionen für den Ersetzungsvorgang aus.
              5. Klicken Sie auf Weitersuchen.
              6. Klicken Sie auf der Symbolleiste Suchen/Ersetzen auf die entsprechende Schaltfläche.

              Verschieben, Kopieren oder Löschen von Text
              1. Markieren Sie den Text, den Sie verschieben, kopieren oder löschen möchten.
              2. Um Text zu verschieben oder zu löschen, klicken Sie auf die Schaltfläche Ausschneiden.
                Um Text zu kopieren, klicken Sie auf die Schaltfläche Kopieren.
              3. Um die Informationen einzufügen, bewegen Sie die Einfügemarke an die gewünschte Stelle, und klicken Sie dann auf die Schaltfläche Einfügen.
              Anmerkungen
              • Um eine Aktion rückgängig zu machen, klicken Sie auf Bearbeiten > Rückgängig.
              • Um das Ergebnis der Aktion wiederherzustellen, klicken Sie auf Bearbeiten > Wiederholen.

                Erstellen von Aufzählungen und Nummerierungen
                1. Klicken Sie auf die Schaltfläche Aufzählung oder Nummerierung.
                2. Geben Sie das erste Element ein.
                3. Drücken Sie die EINGABETASTE, geben Sie das nächste Element ein, und wiederholen Sie diesen Vorgang beliebig oft.
                4. Klicken Sie erneut auf die Schaltfläche, um die Listenformatierung zu beenden.

                Anmerkung Um das Nummerierungsformat für die ausgewählte Liste zu ändern, klicken Sie auf Format > Absatz. Wählen Sie aus der Liste Format das gewünschte Format aus.

                            Einfügen von Symbolen
                            1. Platzieren Sie die Einfügemarke an die Stelle, an der Sie das Symbol einfügen möchten.
                            2. Klicken Sie auf Extras > Symbol einfügen.
                            3. Wählen Sie das gewünschte Symbol aus, und klicken Sie dann auf Einfügen.
                            Tipps
                            • Wenn Sie eine andere Schriftart auswählen (oder die Untergruppe einer Schriftart) wird eine andere Symbolgruppe angezeigt.

                            Drucken eines Dokuments
                            1. Klicken Sie auf Datei > Drucken.
                            2. Wählen Sie die gewünschten Druckoptionen aus.
                            Tipps
                            • Die Liste Anschluss zeigt die verfügbaren Druckeranschlüsse an.
                            • Wenn in der Liste Anschluss die Option Netzwerk ausgewählt ist, geben Sie im Feld Netzpfad den Pfad zum Netzwerkdrucker ein.

                            Einrichten eines Kennworts für ein Dokument
                            1. Klicken Sie auf Datei > Kennwort.
                            2. Geben Sie das Kennwort ein, und bestätigen Sie es.
                            3. Sie müssen das Dokument speichern (Datei > Speichern), um das Kennwort zu übernehmen.

                              Informationen zur Konvertierung von Dokumenten

                                Wenn auf Ihrem Gerät ein E-Mail-Programm installiert ist und Sie eine E-Mail-Nachricht mit einem angehängten Word-Dokument oder einer Word-Vorlage empfangen und diese Datei in Word, Version 6.0 oder höher, erstellt wurde, konvertiert WordPad die Datei auf Ihrem Gerät. Um das Dokument anzuzeigen, öffnen Sie einfach den Anhang in Ihrem E-Mail-Programm, oder öffnen Sie die Datei in WordPad.

                                  Während der Konvertierung werden einige Formatierungsattribute möglicherweise geändert oder gehen verloren. Um zu verhindern, dass diese Attribute dauerhaft verloren gehen, schließen Sie die Datei, nachdem Sie sie angezeigt haben, anstatt sie auf dem mobilen Gerät zu speichern. Wenn Sie die Datei speichern, speichern Sie sie unter einem anderen Namen.

                                    Weitere Informationen zur Konvertierung, die auf Ihrem mobilen Gerät durchgeführt wird, finden Sie in den folgenden Themen.

                                        Formatierung

                                          Ränder und Schattierungen
                                          Ränder und Schattierungen werden nicht in WordPad angezeigt; sie werden jedoch wiederhergestellt, wenn die Datei erneut in ein Word-Dokument konvertiert wird. In Tabellen werden nur einfache Ränder, aber keine Schattierungen, wiederhergestellt.

                                            Schriftschnitt
                                            Fetter, kursiver, durchgestrichener, hochgestellter, tiefgestellter und verborgener Text wird beibehalten und angezeigt. Andere Effekte werden geändert oder entfernt.

                                              Farben
                                              Farben werden beibehalten und als Farben, die auf dem mobilen Gerät verfügbar sind, wiedergegeben.

                                                Schriftarten und -größen
                                                Schriftarten, die nicht vom mobilen Gerät unterstützt werden, werden in einer möglichst ähnlichen, in WordPad verfügbaren Schriftart wiedergegeben.

                                                  Seitenformatierung
                                                  Kopfzeilen, Fußzeilen, Fußnoten, Spalten, Informationen zur Seiteneinrichtung und Seitenformatsinformationen werden bei der Konvertierung in ein WordPad-Dokument entfernt. WordPad unterstützt integrierte Überschriften.

                                                    Absatzformatierung
                                                    Tabstopps, Ausrichtungen, Aufzählungen, Einzüge, einfache Nummerierungen und Absatzabstände werden in Word Pad beibehalten und angezeigt.


                                                      Bilder

                                                        Bilder werden bei der Konvertierung entfernt.


                                                        Inhaltsverzeichnis

                                                          Text aus Inhaltsverzeichnissen und einige Formatierungen werden beibehalten. Text mit rechter Tabstopp-Ausrichtung wird jedoch möglicherweise umgebrochen und in die nächste Zeile geschrieben.


                                                          Index

                                                            Indextext und einige Formatierungen werden beibehalten.


                                                            Tabellen

                                                              Tabellen werden als Text angezeigt, der durch Tabstopps begrenzt ist. Zellen mit umgebrochenem Text, Tabstopps oder Absatzmarken sind möglicherweise schlecht lesbar. Einfache Tabellen werden korrekt angezeigt und bei der Rückkonvertierung der WordPad-Datei in ein Word-Dokument wiederhergestellt.


                                                              OLE-Objekte

                                                                OLE-Objekte werden bei der Konvertierung entfernt.


                                                                Bearbeitungsmarken, Anmerkungen und Kommentare

                                                                  Bearbeitungsmarken werden angezeigt, können aber nicht geändert werden. Anmerkungen und Kommentare werden bei der Konvertierung entfernt.


                                                                  Tastenkombinationen
                                                                    Um die folgenden Aktionen auszuführen Drücken Sie
                                                                    Markierten Text fett formatieren STRG+B
                                                                    Markierten Text kursiv formatieren STRG+I
                                                                    Markierten Text unterstreichen STRG+U
                                                                    Absatz links ausrichten STRG+L
                                                                    Absatz zentriert ausrichten STRG+E

                                                                    Absatz als Aufzählung formatieren STRG+UMSCHALT+L
                                                                    Absatz als Nummerierung formatieren STRG+UMSCHALT+J

                                                                    Einzug verringern STRG+UMSCHALT+M
                                                                    Hängenden Einzug vergrößern STRG+ T
                                                                    Hängenden Einzug verringern STRG+UMSCHALT+T
                                                                    Zoom ALT+Z
                                                                    Zoom umkehren ALT+UMSCHALT+Z
                                                                    Einen Absatz aufwärts bewegen STRG+NACH-OBEN-PFEIL
                                                                    Einen Absatz abwärts bewegen STRG+NACH-UNTEN-PFEIL
                                                                    An den Anfang der Ansicht bewegen STRG+ALT+NACH-OBEN-PFEIL
                                                                    An das Ende der Ansicht bewegen STRG+ALT+NACH-UNTEN-PFEIL

                                                                    Verringern des Schriftgrades STRG+UMSCHALT+<
                                                                    Vergrößern des Schriftgrades STRG+UMSCHALT+>
                                                                    Entfernen des Schriftschnitts STRG+UMSCHALT+Z
                                                                    Entfernen des Absatzformats STRG+Q
                                                                    Gliederungsansicht, alle Titel vom Titel X-Format anzeigen ALT+UMSCHALT+X (X ist die Zahl für die Titelebene)
                                                                    Gliederungsansicht, gesamten Text oder alle Titel anzeigen ALT+UMSCHALT+A
                                                                    Gliederungsansicht, Absätze aufwärts und abwärts bewegen ALT+UMSCHALT+NACH-OBEN-PFEIL{ und ALT+UMSCHALT+NACH-UNTEN-PFEIL}
