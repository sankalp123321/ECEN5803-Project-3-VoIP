    내게 필요한 옵션

      제어판의 내게 필요한 옵션을 사용하여 키보드, 디스플레이, 마우스의 작동 방식을 사용자 지정할 수 있습니다.

      제어판의 내게 필요한 옵션에서 사용할 수 있는 내게 필요한 옵션 도구는 다음과 같은 다양한 기능을 수행합니다.

      고정키
      토글키
      소리 탐지
      고대비
      마우스키

      참고 내게 필요한 옵션을 열려면 시작 > 설정 > 제어판을 선택하고 내게 필요한 옵션을 두 번 클릭합니다.


        고정키

          고정키는 키를 한 번에 키를 하나씩 누르면서 동시 키 입력을 가능하게 합니다.

          고정키는 두 개 이상의 키를 동시에 누르기가 어려운 사용자를 위해 개발되었습니다. 바로 가기는 Ctrl+P와 같은 키 조합이 필요하지만, 고정키는 Alt 키, Ctrl 키, Shift 키 등의 보조키 또는 Windows 로고 키를 눌렀을 때 다른 키를 누를 때까지 키 누름이 활성 상태로 유지되도록 합니다.

          고정키를 켜려면

          1. 내게 필요한 옵션을 엽니다. 시작 > 설정 > 제어판을 선택한 다음 내게 필요한 옵션을 두 번 클릭합니다.
          2. 키보드 탭의 고정키에서 고정키 사용 확인란을 선택합니다.

          고정키를 끄려면 고정키 사용 확인란의 선택을 취소합니다.

          참고 고정키 설정 대화 상자의 바로 가기 사용 확인란을 선택하면 Shift 키를 다섯 번 눌러서 고정키를 켜거나 끌 수 있습니다.

            고정키 옵션 변경

              고정키는 두 개 이상의 키를 동시에 누르기가 어려운 사용자를 위해 개발되었습니다. 바로 가기는 Ctrl+P와 같은 키 조합이 필요하지만, 고정키는 Alt 키, Ctrl 키, Shift 키 등의 보조키 또는 Windows 로고 키를 눌렀을 때 다른 키를 누를 때까지 키 누름이 활성 상태로 유지되도록 합니다.

              1. 내게 필요한 옵션을 엽니다. 시작 > 설정 > 제어판을 선택한 다음 내게 필요한 옵션을 두 번 클릭합니다.
              2. 키보드 탭의 고정키에서 설정을 클릭합니다. 다음 중 하나를 수행합니다.
                • Shift 키를 다섯 번 눌러서 고정키를 끄고 켤 수 있도록 지정하려면 바로 가기 사용 확인란을 선택합니다.
                • Alt 키, Ctrl 키, Shift 키 등의 보조키 또는 Windows 로고 키를 연속해서 두 번 누를 경우, 다시 누를 때까지 키 누름이 활성 상태로 유지되도록 지정하려면 옵션 및 알림에서 보조키를 두 번 눌러 고정 확인란을 선택합니다.
                • Alt 키, Ctrl 키, Shift 키 등의 보조키 또는 Windows 로고 키를 누르거나 잠그거나 놓을 때 소리가 나도록 하려면 옵션 및 알림에서 보조키를 누를 때 신호음 울림 확인란을 선택합니다.

              토글키

                토글키는 잠금키가 눌러져 있을 때 소리를 냅니다.

                토글키는 시각 장애나 인지 장애가 있는 사용자를 위해 개발되었습니다. 토글키를 켠 상태에서 Caps Lock 키, Num Lock 키, Scroll Lock 키와 같은 잠금키를 누르면 컴퓨터에서 소리가 납니다. 잠금키를 켤 때는 고음이 나고, 잠금키를 끌 때는 저음이 납니다.

                토글키를 켜려면

                1. 내게 필요한 옵션을 엽니다. 시작 > 설정 > 제어판을 선택한 다음 내게 필요한 옵션을 두 번 클릭합니다.
                2. 키보드 탭의 토글키에서 토글키 사용 확인란을 선택합니다.

                토글키를 끄려면 토글키 사용 확인란의 선택을 취소합니다.

                참고
                • Num Lock 키를 8초 동안 눌러서 토글키를 켜거나 끌 수 있도록 키보드 바로 가기를 사용하려면 키보드 탭의 토글키에서 설정을 클릭합니다.
                • 토글키 설정 대화 상자에서 바로 가기 사용 확인란을 선택하면 Num Lock 키를 8초 동안 눌러서 토글키를 켜거나 끌 수 있습니다.

                소리 탐지

                  소리 탐지는 시스템 소리에 대한 시각적인 경고를 제공합니다.

                  소리 탐지는 청각 장애로 인해 컴퓨터에서 발생하는 시스템 소리를 잘 듣지 못하는 사용자를 위해 개발되었습니다. 소리 탐지 기능을 사용하면 컴퓨터에서 소리를 낼 때마다 제목 표시줄이 깜박이거나 테두리가 반짝이는 등 시각적 경고가 발생합니다.

                  소리 탐지를 켜려면

                  1. 내게 필요한 옵션을 엽니다. 시작 > 설정 > 제어판을 선택한 다음 내게 필요한 옵션을 두 번 클릭합니다.
                  2. 소리 탭의 소리 탐지에서 소리 탐지 사용 확인란을 선택합니다.

                  소리 탐지를 끄려면 소리 탐지 사용 확인란의 선택을 취소합니다.


                  고대비

                    고대비는 대체 색과 글꼴 크기로 화면 대비를 향상시킵니다.

                    고대비는 시각 장애가 있는 사용자를 위해 개발되었습니다. 고대비 색 구성표는 대체 색 조합을 사용하여 화면 대비를 높임으로써 일부 사용자가 화면을 좀 더 쉽게 볼 수 있도록 합니다. 구성표 중 일부는 읽기 쉽도록 글꼴 크기도 바꿉니다.

                    고대비를 켜려면

                    1. 내게 필요한 옵션을 엽니다. 시작 > 설정 > 제어판을 선택한 다음 내게 필요한 옵션을 두 번 클릭합니다.
                    2. 디스플레이 탭의 고대비에서 고대비 사용 확인란을 선택합니다.

                    고대비를 끄려면 고대비 사용 확인란의 선택을 취소합니다.

                    참고 고대비 설정 대화 상자에서 바로 가기 사용을 선택한 경우에는 왼쪽 Alt+왼쪽 Shift+Print Screen(내게 필요한 옵션 대화 상자에서 선택한 기타 설정에 따라)을 눌러 고대비를 켜거나 끌 수 있습니다. 고대비 설정 대화 상자를 열려면 내게 필요한 옵션을 열고 디스플레이 탭을 선택한 다음 고대비에서 설정을 클릭합니다.

                        마우스키

                          마우스키는 키보드로 마우스 기능을 수행할 수 있게 합니다.

                          마우스키는 마우스 사용에 불편이 있는 사용자를 위해 개발되었습니다. 마우스키를 사용하면 숫자 키패드를 사용하여 마우스 포인터를 제어할 수 있습니다. 숫자 키패드를 사용하여 탐색뿐 아니라 데이터도 입력하려면 Num Lock 키를 눌러 마우스키를 활성화합니다.

                          1. 내게 필요한 옵션을 엽니다. 시작 > 설정 > 제어판을 선택한 다음 내게 필요한 옵션을 두 번 클릭합니다.
                          2. 마우스 탭의 마우스키에서 설정을 클릭합니다.
                          3. 필요한 경우 다음 작업을 완료합니다.
                            • 바로 가기 사용 확인란
                              바로 가기(왼쪽 Alt+왼쪽 Shift+Num Lock)를 눌러 마우스키를 켜거나 끄려면 이 확인란을 선택합니다.
                            • 최고 속도 슬라이더
                              마우스키의 방향키를 누를 때 마우스 포인터가 움직이는 최대 속도를 설정하려면 이 슬라이더를 끌어 줍니다.
                            • 가속도 슬라이더
                              마우스키의 방향키를 누를 때 마우스 포인터가 얼마나 빨리 최대 속도로 가속되는지 설정하려면 이 슬라이더를 끌어 줍니다.
                            • 속도를 높이려면 <Ctrl> 키, 낮추려면 <Shift> 키 누름 확인란
                              이 확인란을 선택하면 Shift 키를 누른 상태에서 마우스키의 방향키를 누를 경우 포인터가 천천히 움직이고, Ctrl 키를 누른 상태에서 마우스키의 방향키를 누를 경우 포인터가 빨리 움직입니다.
                            • 마우스키를 사용할 수 있는 Num Lock 상태
                              해제: 데이터 입력을 위해 키패드를 사용하고 Num Lock을 눌러 마우스키와 표준 숫자 키패드 사이를 전환하는 경우
                              설정: 주로 탐색을 위해 숫자 키패드를 사용하는 경우

                          마우스키 켜기

                            마우스키는 마우스 사용에 불편이 있는 사용자를 위해 개발되었습니다. 마우스키를 사용하면 숫자 키패드를 사용하여 마우스 포인터를 제어할 수 있습니다. 숫자 키패드를 사용하여 탐색뿐 아니라 데이터도 입력하려면 Num Lock 키를 눌러 마우스키를 활성화합니다.

                            마우스키를 켜려면

                            1. 내게 필요한 옵션을 엽니다.
                            2. 마우스 탭의 마우스키에서 마우스키 사용 확인란을 선택합니다.

                            마우스키를 끄려면 마우스키 사용 확인란의 선택을 취소합니다.

                            참고 항목
                            키 입력을 사용한 마우스키 켜기

                            키 입력을 사용한 마우스키 켜기

                              1. 왼쪽 Alt + 왼쪽 Shift + Num Lock을 누릅니다.
                              2. 마우스키 메시지 대화 상자에서 스페이스바를 눌러 내게 필요한 옵션 기능의 바로 가기 키 설정 해제 확인란의 선택을 취소합니다.
                              3. Tab 키를 누르고 Enter 키를 누릅니다.

                              위의 키 입력으로 마우스키가 켜지지 않으면 다음 단계를 수행합니다.

                              1. 내게 필요한 옵션을 엽니다.
                              2. 마우스 탭이 선택될 때까지 Ctrl+Tab을 누릅니다.
                              3. 마우스키 사용을 선택하려면 스페이스바를 누릅니다.
                              4. Tab 키를 두 번 눌러 확인을 선택하고 Enter 키를 누릅니다.
                              참고
                              • 마우스키의 설정을 변경하려면 마우스 탭에서 설정을 클릭합니다.
                              • 마우스키 설정 대화 상자에서 바로 가기 사용 확인란을 선택하면 왼쪽 Alt+왼쪽 Shift+Num Lock을 눌러 마우스키를 켜거나 끌 수 있습니다.

                              마우스키를 사용한 클릭

                                1. 마우스키를 켭니다. 마우스키를 켜려면 내게 필요한 옵션을 열고 마우스 탭을 선택한 다음, 마우스키 사용 확인란을 선택합니다.
                                2. 숫자 키패드의 단추를 사용하여 클릭, 두 번 클릭, 마우스 오른쪽 단추를 클릭할 수 있습니다. 마우스키가 작동하지 않으면 Num Lock 키를 누르고 다시 마우스키를 사용해 봅니다.

                                  중요 !! 숫자 키패드에서 모든 마우스키 동작을 수행합니다.

                                • 왼쪽 클릭
                                    마우스 왼쪽 단추를 한 번 클릭하려면 나누기 기호(/) 키를 눌러 왼쪽을 활성화한 다음 5를 누릅니다.
                                  • 마우스 왼쪽 단추를 두 번 클릭하려면 나누기 기호(/) 키를 눌러 왼쪽을 활성화한 다음 더하기 기호(+)를 누릅니다.
                                • 오른쪽 클릭
                                  • 마우스 오른쪽 단추를 한 번 클릭하려면 빼기 기호(-)를 눌러 오른쪽을 활성화한 다음 5를 누릅니다.
                                  • 마우스 오른쪽 단추를 두 번 클릭하려면 빼기 기호(-)를 눌러 오른쪽을 활성화한 다음 더하기 기호(+)를 누릅니다.
                                참고

                                                                                  배터리 수명 최적화

                                                                                    특히 이동하면서 사용하는 경우에는 배터리를 가능한 한 오래 지속시켜야 합니다. 일반적인 조건에서는 배터리 하나로도 장시간 사용할 수 있습니다. 다음은 배터리를 가장 오래 사용할 수 있는 방법입니다.

                                                                                      • 가능하면 외부 전원을 사용합니다.
                                                                                        다음과 같은 경우 가능하면 AC 어댑터를 사용하여 장치를 외부 전원에 연결합니다.
                                                                                        • 전화 접속 연결로 설정한 경우(일부 모뎀은 전원을 많이 사용합니다.)
                                                                                        • 장치 디스플레이의 조명 기능을 사용하는 경우
                                                                                        • 데스크톱 컴퓨터에 연결한 경우
                                                                                        • 소형 플래시 카드 또는 기타 주변 장치를 사용하는 경우
                                                                                        • 확장 시간 동안 LED 표시기가 깜박이도록 한 경우
                                                                                      • 유휴 상태일 때 장치를 끄도록 설정합니다.
                                                                                        배터리 전원을 사용할 경우 3분 동안 키를 입력하지 않거나 스타일러스나 마우스를 사용하지 않으면 장치 전원이 자동으로 꺼지거나 작업이 일시 중단됩니다. 이 시간을 줄여 배터리 수명을 최대화할 수 있습니다.
                                                                                      • 필요하지 않으면 소리를 끕니다.
                                                                                        기본적으로 장치는 경고나 약속, 하드웨어 단추 누름과 같은 많은 이벤트에 대한 응답으로 소리를 냅니다. 배터리 수명을 최적화하려면 필요 없는 소리는 모두 끕니다. 시작, 설정, 제어판을 선택하고 볼륨과 소리를 선택합니다.

                                                                                        시간을 "안 함"으로 설정해도 상황에 따라 시스템이 전원 절약 상태가 될 수 있습니다. 예를 들어 "일시 중단 상태로 상태 전환:"을 "안 함"으로 설정한 경우 배터리가 위험 수준에 도달하면 시스템이 일시 중단될 수 있습니다.

                                                                                        팁 시간이 짧을수록 배터리 전원이 절약됩니다.

                                                                                        1. 시작 > 설정 > 제어판을 선택합니다.
                                                                                        2. 전원 아이콘을 연 다음 장치 상태 탭을 선택합니다.

                                                                                        참고 수동형 전원 관리를 사용하도록 설정한 경우, 장치 상태 탭이 나타나지 않습니다. 수동형 전원 관리 설정에 대한 자세한 내용은 유휴 상태일 때 장치 전원 끄기 설정을 참조하십시오.

                                                                                                                            선택한 저장소 장치에서 사용 가능한 파티션이 목록 상자에 나열됩니다.

                                                                                                                            • 저장소 장치에 새 파티션을 만들려면 새로 만들기를 선택합니다.
                                                                                                                            • 저장소 장치에서 파티션을 삭제하려면 삭제를 선택합니다. 탑재된 파티션은 삭제할 수 없습니다.
                                                                                                                            • 파티션 정보를 보거나 포맷 등의 고급 파티션 기능을 수행하려면 속성을 선택합니다.
                                                                                                                            참고 항목
                                                                                                                            새 파티션 작성
                                                                                                                            고급 파티션 기능

                                                                                                                            새 파티션 작성

                                                                                                                              1. 새 파티션을 작성하려면 목록에서 저장소 장치를 선택하고 새로 만들기를 선택합니다.
                                                                                                                              2. 새 파티션의 이름을 입력합니다.
                                                                                                                              3. 새 파티션의 크기(섹터)를 입력하거나 사용 가능한 디스크 공간 모두 사용 상자를 선택합니다.
                                                                                                                              4. 파티션을 만들려면 확인을 선택하고 취소하려면 X를 선택합니다.

                                                                                                                              고급 파티션 기능

                                                                                                                                고급 파티션 옵션을 보려면 파티션을 선택하고 속성을 선택합니다.

                                                                                                                                • 분리된 파티션을 탑재하려면 탑재를 선택합니다.
                                                                                                                                • 탑재된 파티션을 분리하려면 분리를 선택합니다.
                                                                                                                                • 파티션을 포맷하려면 포맷을 선택합니다.
                                                                                                                                • 파티션을 스캔 및 복구하려면 스캔을 선택합니다.
                                                                                                                                • 파티션에서 조각 모음을 실행하려면 조각 모음을 선택합니다.
                                                                                                                                참고
                                                                                                                                • 파티션의 파일 시스템에 따라 포맷, 스캔 및 조각 모음 옵션을 사용하지 못할 수 있습니다. 이러한 기능의 동작은 파일 시스템의 유틸리티 라이브러리 구현에 따라 다릅니다.

                                                                                                                                    Bluetooth 장치 검색

                                                                                                                                      1. 시작 > 설정 > 제어판을 선택한 다음 Bluetooth를 두 번 클릭합니다.
                                                                                                                                      2. 장치 검색 단추를 누릅니다. 조회 과정이 시작됩니다. 검색을 통해 근접한 위치에서 Bluetooth 장치가 발견되면 그 장치가 신뢰되지 않음 목록 상자에 추가됩니다.
                                                                                                                                      3. 그러면 신뢰 관계를 형성할 장치를 선택할 수 있습니다. 신뢰 관계를 위한 장치 인증에 대한 자세한 내용은 장치 인증을 참조하십시오.

                                                                                                                                      장치 인증
                                                                                                                                      신뢰됨 목록에서 장치 제거
                                                                                                                                      장치 활성화

                                                                                                                                      장치 인증

                                                                                                                                        1. 시작 > 설정 > 제어판을 선택한 다음 Bluetooth를 두 번 클릭합니다.
                                                                                                                                        2. 신뢰되지 않음 목록 상자에서 인증할 장치를 클릭합니다.
                                                                                                                                        3. "-->" 단추를 클릭합니다. 장치에 대한 인증이 필요한 경우, 장치의 개인 ID 번호(PIN)를 묻는 대화 상자가 나타납니다. 인증이 성공하면 신뢰됨 목록 상자로 장치가 이동합니다.

                                                                                                                                        신뢰됨 목록에서 장치 제거

                                                                                                                                          1. 시작 > 설정 > 제어판을 선택한 다음 Bluetooth를 두 번 클릭합니다.
                                                                                                                                          2. 신뢰됨 목록 상자에서 제거할 장치를 클릭합니다.
                                                                                                                                          3. "<--" 단추를 클릭하여 해당 장치를 신뢰되지 않음으로 설정합니다.
                                                                                                                                          4. 항목을 삭제하고 캐시에서 제거하려면 Del 단추를 클릭합니다.
