    Avvio dei programmi

      Per aprire un programma che non è presente sul desktop, fare clic sul pulsante Start, scegliere Programmi, quindi il nome del programma.


      Uso della Guida contestuale
      • Per informazioni su un programma o una finestra di dialogo, scegliere il pulsante ? quando si utilizza il programma o la finestra di dialogo.
      • Per visualizzare il nome di un pulsante della barra degli strumenti, toccare il pulsante con lo stilo e mantenere lo stilo in posizione. Per evitare che il comando del pulsante venga eseguito, impugnare lo stilo mentre lo si trascina al di fuori della superficie del pulsante.
      • Per visualizzare il nome di un pulsante della barra degli strumenti utilizzando il mouse, spostare il puntatore del mouse sul pulsante interessato.

      Apertura di un menu di scelta rapida
      • Se si utilizza lo stilo, toccare il tasto ALT, quindi l'elemento interessato.
      • Se si utilizza il mouse, fare clic con il pulsante destro sull'elemento.

              Passaggio da un menu all'altro e da un pulsante della barra degli strumenti all'altro

                Utilizzare il dispositivo di scorrimento per controllare la visualizzazione dei menu e della barra degli strumenti.

                • Per passare rapidamente da un menu all'altro e da un pulsante della barra degli strumenti all'altro, selezionare il dispositivo di scorrimento.
                • Trascinare il dispositivo di scorrimento verso sinistra o verso destra per visualizzare i menu, i pulsanti della barra degli strumenti o una combinazione di entrambi.
                • Per visualizzare sia i menu che i pulsanti della barra degli strumenti, trascinare il dispositivo di scorrimento al di sotto dei menu.

                2. Scegliere Salva con nome dal menu File.
                3. Salvare il file nella cartella appositamente denominata per la scheda di memoria.

                        1. Se necessario, aprire la cartella in cui si desidera posizionare la nuova cartella.
                        2. Scegliere Nuova cartella dal menu File.

                        Aggiunta di una cartella all'elenco Preferiti
                        1. Aprire la cartella.
                        2. Scegliere Aggiungi a Preferiti dal menu Preferiti.

                        • Per ricevere un file, allineare le porte a infrarossi e scegliere Ricevi dal menu File.

                        Creazione di collegamenti
                        1. Visualizzare il file o la pagina Web per i quali si desidera creare un collegamento.
                        2. Scegliere Invia a dal menu File.
                        3. Scegliere Collegamento sul desktop.

                            Informazioni su Pocket Internet Explorer

                              Con Pocket Internet Explorer è possibile visualizzare sul dispositivo le pagine Web di Internet o di una rete Intranet. È necessario utilizzare un modem o una scheda Ethernet per eseguire la connessione a un ISP (Internet Service Provider) o in rete.

                                  Connessione a Internet

                                    Prima di poter visualizzare le pagine Web, è necessario impostare una connessione remota all'ISP o alla rete. Se non è stata impostata una connessione all'ISP o alla rete, vedere Connessioni remote.

                                    1. Fare clic sul pulsante Start, scegliere Programmi, quindi Comunicazioni e infine Accesso remoto. Toccare due volte o fare doppio clic sulla connessione che si desidera utilizzare.
                                    2. Immettere tutte le informazioni sull'utente richieste. Per modificare la modalità di composizione del numero, scegliere il pulsante Proprietà.
                                    3. Scegliere il pulsante Connetti.

                                    Nota Se la connessione presenta dei problemi, potrebbe essere necessario accedere a Internet tramite un server proxy.

                                      Vedere anche
                                      Impostazione di un server proxy

                                      Impostazione di una connessione automatica a Internet

                                        Se si imposta una connessione automatica a Internet, la connessione remota scelta verrà composta automaticamente quando Pocket Internet Explorer deve accedere a Internet o alla rete aziendale.

                                        1. Scegliere Opzioni dal menu Visualizza, quindi fare clic sulla scheda Connessione automatica.
                                        2. Selezionare la casella di controllo Connetti a Internet quando necessario.
                                        3. Specificare la connessione remota da utilizzare.
                                        4. Se si desidera eseguire la disconnessione automatica da Internet dopo un certo periodo di inattività (durante il quale il dispositivo non è stato utilizzato), selezionare la casella di controllo Disconnessione automatica.
                                        5. Nella casella Disconnetti se inattiva per immettere il periodo di inattività al termine del quale il dispositivo verrà disconnesso da Internet.
                                        Vedere anche
                                        Connessione a Internet

                                        Impostazione di un server proxy

                                          I server proxy vengono utilizzati spesso per una protezione ulteriore nelle connessioni a Internet tramite una rete locale, quale una rete aziendale.

                                          1. Scegliere Opzioni dal menu Visualizza, quindi fare clic sulla scheda Server proxy.
                                          2. Selezionare la casella di controllo Usa server proxy.
                                          3. Immettere l'indirizzo e la porta del server proxy. Per ulteriori informazioni, rivolgersi all'amministratore della rete.
                                          4. Per ignorare il server proxy per gli indirizzi locali, quali le pagine della rete Intranet aziendale, selezionare la casella di controllo Ignora server proxy per indirizzi locali.

                                          Accettazione dei cookie

                                            Un file cookie contiene informazioni relative all'identità e alle preferenze dell'utente che consentono di personalizzare, in base alle esigenze specifiche, le informazioni di un sito Web. Il sito invia il file, che viene memorizzato nel dispositivo portatile.

                                            1. Scegliere Opzioni dal menu Visualizza, quindi fare clic sulla scheda Avanzate.
                                            2. Selezionare la casella di controllo Accetta cookie.

                                                    Salvataggio di una copia della pagina corrente

                                                      È possibile salvare le pagine Web e visualizzarle successivamente al termine della connessione a Internet o a una rete Intranet senza utilizzare la memoria cache.

                                                      1. Scegliere Salva con nome dal menu File.
                                                      2. Specificare una posizione di cartella e assegnare un nome al file.

                                                      Note

                                                      • Nelle pagine salvate le immagini non verranno visualizzate e i collegamenti saranno disattivati.
                                                      • Le pagine Web contenenti i frame non verranno visualizzate o ne verrà visualizzata solo la struttura.
                                                      • Per salvare un singolo frame, premere ALT e toccare o fare clic sul frame interessato. Scegliere Salva con nome o Salva immagine con nome dal menu di scelta rapida. Alcuni frame sono progettati in modo da non poter essere salvati. Questi frame non verranno visualizzati quando si tenterà di aprirli come file salvati in Esplora risorse.

                                                      Vedere anche
                                                      Visualizzazione non in linea di una pagina memorizzata nella cache

                                                      Visualizzazione non in linea di una pagina memorizzata nella cache

                                                        Le pagine visualizzate in Internet vengono memorizzate nella cache nella cartella File temporanei Internet in Risorse del computer. È più veloce sfogliare le pagine memorizzate nella cache che tornare ai siti Internet.

                                                        1. Prima di disconnettersi, scegliere Non in linea dal menu File.
                                                        2. Al termine della connessione, scegliere Cronologia dal menu Vai.
                                                        3. Selezionare la pagina che si desidera visualizzare.

                                                        Note

                                                        • I collegamenti verranno disattivati.
                                                        • Durante la visualizzazione delle pagine Web memorizzate nella cache, verrà visualizzato solo il primo di ciascun set di frame. La struttura di alcuni frame non ne consente la memorizzazione nella cache e la successiva visualizzazione quando vengono aperti come file memorizzati nella cache.

                                                        Modifica delle impostazioni per la memoria cache

                                                          In base all'impostazione predefinita, la cache di Pocket Internet Explorer rappresenta il 10% della memoria complessiva del dispositivo. In caso di problemi con la memoria, provare a disattivare la cache di Pocket Internet Explorer.

                                                          1. Scegliere Opzioni dal menu Visualizza, quindi fare clic sulla scheda Avanzate.
                                                          2. Se si desidera attivare il caching (memoria temporanea), selezionare la casella di controllo Attiva cache.
                                                            Il caching comporta un notevole utilizzo della memoria.
                                                          3. Spostare il cursore della casella Dimensione cache per aumentare o diminuire la quantità di memoria da destinare alla cache.
                                                          4. Se si desidera che la cartella venga svuotata alla chiusura di Pocket Internet Explorer, selezionare la casella di controllo Svuota cache all'uscita.
                                                          5. Per svuotare la cartella immediatamente, scegliere il pulsante Svuota cache subito.

                                                            Informazioni sulla protezione

                                                              Pocket Internet Explorer supporta i protocolli di protezione utilizzati per proteggere i siti. Quando si accede a un sito protetto, è possibile inviare senza rischi informazioni di carattere personale. Per stabilire se un sito Web è protetto, controllare che sul pulsante di Pocket Internet Explorer sulla barra di stato sia raffigurata l'icona di una serratura.

                                                                Vedere anche
                                                                Selezione delle opzioni di protezione

                                                                Selezione delle opzioni di protezione
                                                                1. Scegliere Opzioni dal menu Visualizza, quindi fare clic sulla scheda Avanzate.
                                                                2. Scegliere il pulsante Impostazioni protezione.
                                                                3. Selezionare i protocolli di protezione desiderati.
                                                                  • PCT è l'acronimo di Private Communications Technology, un protocollo di proprietà esclusiva sviluppato da Microsoft che garantisce una maggiore protezione rispetto all'SSL2. Il sistema operativo Windows Embedded Compact non supporta il protocollo PCT.
                                                                  • SSL2 è l'acronimo di Secured Sockets Layer Level 2, un protocollo standard utilizzato per le transazioni protette. Tutti i siti Internet protetti supportano questo protocollo.
                                                                  • SSL3 è l'acronimo di Secured Sockets Layer Level 3, un protocollo di proprietà esclusiva destinato a garantire una maggiore protezione rispetto all'SSL2. Alcuni siti Internet non supportano questo protocollo.

                                                                        Protezione delle informazioni importanti

                                                                          È possibile conservare nel dispositivo dati estremamente delicati, quali numeri di telefono di contatti importanti, numeri di carte di credito e documenti riservati. Di seguito vengono elencati alcuni metodi che consentono di proteggere questo tipo di informazioni.

                                                                            • Utilizzare una password di protezione del dispositivo.
                                                                              È possibile impostare una password per far sì che l'accesso al dispositivo sia autorizzato solo per le persone in grado di fornirla. Selezionare Start > Impostazioni > Pannello di controllo. Fare doppio clic sull'icona Password.

                                                                              IMPORTANTE
                                                                              Scegliere una password facile da ricordare: nel caso la si dimenticasse, per utilizzare nuovamente il dispositivo sarebbe necessario eseguire una reimpostazione completa. Tale operazione comporterebbe la cancellazione di tutti i dati esistenti.

                                                                            • Effettuare una copia di backup dei dati.
                                                                              Si consiglia di creare e aggiornare periodicamente un file di backup che includa i dati di programma del dispositivo. Se ad esempio si rimuovessero accidentalmente la batteria principale e quella di backup, con conseguente perdita dei dati, sarà possibile utilizzare il file di backup per ripristinare i dati andati perduti.

                                                                                                                              Finestra di dialogo Colore

                                                                                                                                Colori di base
                                                                                                                                Mostra i colori di base disponibili. Per definire un colore personalizzato, selezionare il colore di base più simile, quindi scegliere Definisci.

                                                                                                                                  Colori personalizzati
                                                                                                                                  Visualizza tutti i colori personalizzati definiti.
                                                                                                                                  Per modificare le impostazioni relative a un colore personalizzato, fare clic su una tavolozza personalizzata, quindi selezionare un colore diverso.
                                                                                                                                  Per definire un nuovo colore personalizzato, fare clic su una tavolozza personalizzata vuota, scegliere un colore diverso, quindi scegliere Agg. personalizzati.

                                                                                                                                    Definisci
                                                                                                                                    Definisce un colore personalizzato. Se questo pulsante non è disponibile, significa che i colori personalizzati sono già visualizzati.
                                                                                                                                    Per definire un colore personalizzato, effettuare la selezione nella matrice dei colori oppure selezionare il colore utilizzando le definizioni della tonalità, della saturazione e della luminosità o le definizioni del rosso, del verde e del blu.

                                                                                                                                      Agg. personalizzati
                                                                                                                                      Aggiunge il colore personalizzato definito.

                                                                                                                                        Colore/Tinta unita
                                                                                                                                        Nella parte sinistra, questa casella visualizza la quantità di bianco e nero contenuta nel colore corrente. Nella parte destra, visualizza solo il colore, senza la componente di bianco e nero. Regolare la combinazione utilizzando il cursore posto a destra.
                                                                                                                                        Per selezionare solo il colore senza bianco o nero, premere ALT+O.
                                                                                                                                        Per cambiare il colore, modificare i valori elencati di seguito oppure fare delle prove selezionando e trascinando il cursore sulla matrice dei colori.

                                                                                                                                          Ton
                                                                                                                                          Il valore di una scala di colori. Rosso = 0, Giallo = 60, Verde = 120, Azzurro = 180, Magenta = 200 e Blu = 240.

                                                                                                                                            Sat
                                                                                                                                            Livello di saturazione; la quantità di colore in una tonalità specificata, fino a 240.

                                                                                                                                              Lum
                                                                                                                                              Luminosità. Al variare della luminosità, cambiano anche i valori del rosso, del verde e del blu.

                                                                                                                                                Rosso
                                                                                                                                                La quantità di rosso contenuta nel colore selezionato.

                                                                                                                                                  Verde
                                                                                                                                                  La quantità di verde contenuta nel colore selezionato.

                                                                                                                                                    Blu
                                                                                                                                                    La quantità di blu contenuta nel colore selezionato.

                                                                                                                                                                                      Impostazione di una password per un file
                                                                                                                                                                                      1. Scegliere Password dal menu File.
                                                                                                                                                                                      2. Digitare e verificare la password.
                                                                                                                                                                                      3. È necessario salvare il file (scegliere Salva dal menu File) per conservare la password.

                                                                                                                                                                                      Nota Non è possibile sincronizzare i file protetti da password. È necessario rimuovere la password di protezione nel programma appropriato sul PC o sul dispositivo e quindi eseguire la sincronizzazione. Al termine della sincronizzazione, è possibile reimpostare la password di protezione.
