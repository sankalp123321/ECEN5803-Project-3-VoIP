    WordPad 정보

      Microsoft Word 데스크톱 버전과 비슷한 단추와 메뉴 명령을 사용하여 WordPad에서 문서와 서식 파일을 만들고 편집할 수 있습니다. 기본 보기 또는 개요 보기에서 파일 작업을 할 수 있습니다.

        WordPad 문서는 일반적으로 .pwd 파일로 저장되지만 .rtf 또는 .doc와 같은 다른 파일 형식으로 저장할 수도 있습니다.

            텍스트 선택
              선택 대상 수행 작업
              텍스트 스타일러스 또는 포인터를 텍스트 위로 끕니다.
              단어 단어를 두 번 클릭합니다.
              단일 단락 단락 안에서 세 번 클릭합니다.
              전체 문서 편집 > 모두 선택을 선택합니다.
              그림 해당 그림을 클릭합니다.


              텍스트 찾기
              1. 문서의 일부에서 텍스트를 검색하려면 해당 부분을 선택합니다.
                그렇지 않으면 전체 문서가 검색됩니다.
              2. 편집 > 찾기를 클릭합니다.
              3. 찾을 텍스트를 입력합니다.
                단락에는 ^p를 입력합니다. 탭 정지에는 ^t를 입력합니다.
              4. 원하는 검색 옵션을 선택합니다.
              5. 다음 찾기를 선택합니다.
              6. 검색을 계속하려면 편집 > 다음 찾기를 선택합니다.

              텍스트 바꾸기
              1. 문서의 일부에서 텍스트를 바꾸려면 해당 부분을 선택합니다.
                그렇지 않으면 전체 문서가 검색됩니다.
              2. 편집 > 바꾸기를 선택합니다.
              3. 찾을 텍스트와 바꿀 텍스트를 입력합니다.
                단락에는 ^p를 입력합니다. 탭 정지에는 ^t를 입력합니다.
              4. 원하는 대체 옵션을 선택합니다.
              5. 다음 찾기를 선택합니다.
              6. 찾기/바꾸기 도구 모음에서 해당 단추를 선택합니다.

              1. 이동, 복사 또는 삭제할 텍스트를 선택합니다.
              2. 텍스트를 이동 또는 삭제하려면 잘라내기 단추를 선택합니다.
                텍스트를 복사하려면 복사 단추를 선택합니다.
              3. 정보를 붙여넣으려면 삽입점을 원하는 위치로 이동하고 붙여넣기 단추를 선택합니다.

                              문서 변환 정보

                                장치에 전자 메일 프로그램이 설치되어 있고 Word 6.0 이상에서 작성한 Word 문서 또는 서식 파일을 메시지 첨부 파일로 받는 경우 WordPad는 장치에 있는 파일을 변환합니다. 문서를 보려면 전자 메일 프로그램에서 첨부 파일을 열거나 WordPad에서 해당 파일을 엽니다.

                                  변환하는 동안 여러 서식 속성이 변경되거나 손상될 수 있습니다. 속성을 영구히 잃지 않으려면 파일을 본 다음 파일을 저장하지 않고 닫습니다. 파일을 저장하면 해당 파일이 다른 파일 이름으로 저장됩니다.

                                    장치에서 수행되는 변환에 대한 자세한 내용은 다음 항목을 참조하십시오.

                                        서식 지정

                                          테두리 및 그림자
                                          테두리와 그림자는 WordPad에는 표시되지 않지만 Word 문서로 다시 변환하면 복원됩니다. 표에서 사용할 경우 그림자는 복원되지 않지만 단순 테두리는 복원됩니다.

                                            문자 서식
                                            굵게, 기울임꼴, 취소선, 위 첨자, 아래 첨자, 숨김 텍스트는 유지되고 표시됩니다. 다른 효과는 변경되거나 제거됩니다.

                                              색
                                              색은 유지되어 장치에서 사용 가능한 색으로 매핑됩니다.

                                                글꼴 및 글꼴 크기
                                                이동 장치에서 지원되지 않는 글꼴은 WordPad에서 사용할 수 있는 가장 유사한 글꼴로 매핑됩니다.

                                                  페이지 서식
                                                  머리글, 바닥글, 주석, 열, 페이지 설정 정보, 스타일시트 등은 WordPad로 변환되는 동안 제거됩니다. WordPad는 빌트인 제목을 지원합니다.

                                                    단락 서식
                                                    탭, 맞춤, 글머리 기호, 들여쓰기, 단순한 번호 매기기, 단락 간격 등은 WordPad에서도 표시됩니다.


                                                      그림

                                                        그림은 변환하는 동안 제거됩니다.


                                                        목차

                                                          목차 텍스트와 일부 서식은 유지됩니다.그러나, 오른쪽 맞춤 탭 정지로 설정된 텍스트는 다음 줄로 줄 바꿈될 수 있습니다.


                                                          색인

                                                            색인 텍스트와 일부 서식은 유지됩니다.


                                                            표

                                                              표는 탭 구분 텍스트로 표시됩니다. 보호된 텍스트, 탭, 또는 단락 표식이 포함된 셀은 읽기가 어려울 수 있습니다. 단순한 표는 WordPad 파일을 Word 문서로 다시 변환하면 정확히 표시되고 복원됩니다.


                                                              OLE 개체

                                                                OLE 개체는 변환하는 동안 제거됩니다.


                                                                수정 표시, 주석, 설명

                                                                  수정 표시는 볼 수는 있지만 변경할 수는 없습니다. 주석과 설명은 변환하는 동안 제거됩니다.
