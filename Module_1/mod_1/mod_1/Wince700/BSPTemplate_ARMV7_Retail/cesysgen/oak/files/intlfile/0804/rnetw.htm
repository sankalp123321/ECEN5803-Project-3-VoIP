      关于连接到 Internet

        可以使用移动设备连接到 Internet，浏览万维网 (WWW) 和收发电子邮件。

          如果您的设备没有配备内置调制解调器，则需要安装调制解调器卡或外置调制解调器。有关安装的详细信息，请参阅调制解调器制造商的指导说明。设置好调制解调器后，即可创建与 ISP 的连接。

            创建 ISP 帐户

              注意：您的设备不支持 PPTP 帐户。

                通过 Internet 服务提供商设置点对点协议 (PPP) 帐户并获取以下信息：

                • 访问电话号码
                • 用户名
                • 密码

                建立帐户后，在设备上新建一个连接。创建此连接时，您应可以使用“新建连接”向导中提供的所有默认 TCP/IP 设置。如果使用默认设置无法连接，请与 ISP 联系以获取具体的 TCP/IP 信息以及主 DNS 地址和辅助 DNS 地址。

                  在设备上新建连接

                  在设备上新建连接
                    1. 选择“开始”>“设置”>“网络和拨号连接”。
                    2. 双击“新建连接”图标。
                    3. 在“新建连接”对话框中，输入连接的名称。
                    4. 选择“拨号连接”。
                    5. 选择“下一步”按钮。
                    6. 从下拉列表中选择要使用的调制解调器。
                    7. 如果希望每次连接时都输入新的电话号码，请选择“配置”。
                      • 在“设备属性”对话框中，选择“端口设置”选项卡。
                      • 选中“手动拨号”以便能够在每次连接时都输入电话号码。
                      • 如果您要自定义电话选项，请选择“拨号选项”选项卡并设置首选项。
                      • 选择“确定”以接受您选择的配置。
                    8. 选择“TCP/IP 设置”。
                      • 在“常规”选项卡上，请确保已选中“使用服务器分配的 IP 地址”。
                      • 在“名称服务器”选项卡上，请确保已选中“使用服务器分配的地址”。
                      • 选择“确定”。
                    9. 选择“下一步”按钮。
                    10. 在“电话号码”对话框中，输入 ISP 的电话访问号码。
                    11. 选择“完成”按钮。

                    刚刚创建的连接在“网络连接”面板中显示为一个图标。

                    注意：

                    • 如果使用默认设置无法连接，请咨询您的 ISP 或网络管理员以获得帮助。
                    • 若要为外置调制解调器创建 PC 调制解调器连接，请选择“Hayes Compatible on COM1”作为您的调制解调器。
                    • 使用外置调制解调器将设备连接到 PC 时，请使用调制解调器制造商提供的调制解调器电缆或适配器。

                        关于连接到网络

                          您可以通过网络（以太网）或拨号连接直接连接到网络。您可以访问电子邮件，访问网络服务器上可用的文件，还可以浏览 Internet 或 Intranet。要接收电子邮件，请在创建到网络的连接后，请在收件箱中创建 POP3 或 IMAP4 邮件服务。有关详细信息，请参阅收件箱帮助。

                            创建网络（以太网）连接

                              1. 将以太网卡插入设备。
                              2. 在“以太网驱动程序设置”对话框中，选中“通过 DHCP 获得 IP 地址”。选择“确定”。
                              3. 选择“开始”>“设置”>“控制面板”。双击“网络和拨号连接”图标。
                              4. 双击适配器的连接图标，然后单击“确定”确认设置。
                                例如，若您有一个 DC21x4 以太网适配器，则双击“DC211401”连接图标。
                              5. 系统提示时，输入用于登录到您的网络的“用户名”、“密码”和“域名”。

                                创建拨号网络连接

                                1. 请网络管理员为您创建一个 RAS 帐户。设置好帐户后，请获取下列信息：拨号电话号码、密码以及计算机名。
                                2. 在您的设备上使用步骤 1 中的信息创建新的连接。您应能使用“新建连接”向导中提供的默认 TCP/IP 设置。如果使用这些默认设置无法连接，请联系网络管理员获取特定的 TCP/IP 信息。

                                在设备上新建连接

                                  除非使用以太网卡，否则必须针对要远程访问的每个计算机或网络分别创建一个连接。

                                  1. 选择“开始”>“设置”>“网络和拨号连接”。
                                  2. 双击“新建连接”图标。
                                  3. 在“新建连接”对话框中，输入连接的名称。
                                  4. 选择“拨号连接”并单击“下一步”按钮。
                                  5. 选择要使用的调制解调器。
                                  6. I如果希望在每次连接时输入电话号码，
                                    • 请选择“手动拨号”，然后单击“确定”。
                                    • 选择“TCP/IP 设置”。
                                    • 在“常规”选项卡上，请确保已选中“使用服务器分配的 IP 地址”。
                                    • 在“名称服务器”选项卡上，请确保已选中“使用服务器分配的地址”，然后选择“确定”。
                                    • 选择“下一步”按钮，并键入电话号码。
                                    • 选择“完成”按钮。
                                  刚刚创建的连接在“远程网络”文件夹中显示为一个图标。
                                    注意：
                                    • 如果使用这些默认设置无法连接，请联系 ISP 或网络管理员以获取具体的 TCP/IP 信息。
                                    • 此连接在“连接”屏幕上显示为一个图标。
                                    • 如果您在为外部调制解调器创建 PC 调制解调器连接，请选择“Hayes Compatible on COM1”作为您的调制解调器。 使用外置调制解调器将设备连接到 PC 时，请使用调制解调器制造商提供的调制解调器电缆或适配器。

                                      连接到无线网络
                                      1. 插入您的无线卡。系统提示时，输入制造商提供的驱动程序名称。
                                      2. 一旦加载，系统会弹出一个对话框帮助您连接到可用的无线网络。
                                      3. 选择您要连接的网络的名称，然后单击“连接”。
                                      4. 确认检测到的设置，然后单击“确定”。
                                      5. 如果需要查阅或更改设置，请选择“开始”>“设置”>“网络和拨号连接”。
                                      6. 双击要使用的远程连接图标。
                                      7. 如果您的无线卡驱动程序支持该网络，“无线网络”选项卡就可用，您可在其中调整设置。
                                      8. 使用该选项卡查看、更改或调整设置。

                                        修改连接设置
                                        1. 选择“开始”>“设置”>“网络和拨号连接”。
                                        2. 选择要修改连接设置的图标。
                                        3. 选择“文件”>“属性”，或双击适当的图标。
                                        4. 选择所需的选项。根据连接不同，可能还需其他设置。要进行修改，请选择此图标，并从菜单中选择“高级设置...”。

                                        关闭软件压缩
                                        1. 选择“开始”>“设置”>“网络和拨号连接”。
                                        2. 选择要修改其设置的连接的图标。
                                        3. 选择“文件”>“属性”。
                                        4. 选择“TCP/IP 设置”。
                                        5. 在“常规”选项卡上，清除“使用软件压缩”。

                                        注意：虽然关闭软件压缩可缩短完成连接所需的时间，这样做却降低了数据传输的速度。只有在频繁连接且每次会话中传输的数据量较小时才关闭软件压缩。

                                            1. 选择“开始”>“设置”>“控制面板”。

                                              关于终端

                                                可以将设备连接到需要 TTY 或 VT-100 终端仿真的联机服务或企业服务器上。例如，您可以从布告栏查看和下载文件或收发电子邮件。

                                                  只有与要求终端仿真的服务通信时才使用终端。

                                                    新建终端会话
                                                    1. 选择“开始”>“设置”>“控制面板”>“终端服务器客户端许可证”。
                                                    2. 双击“新建会话”图标。
                                                    3. 在“会话名称”框中，输入会话的名称。
                                                    4. 在“选择调制解调器”列表中，选择调制解调器的名称。
                                                    5. 输入远程计算机的电话号码。
                                                    6. 如果不需要调整仿真属性，选择“确定”进行连接。
                                                    注意
                                                    • 创建的会话在“终端”文件夹中显示为一个图标。
                                                    • 若要断开连接，请选择“文件”>“取消”。

                                                    使用现有会话进行连接
                                                    • 在终端窗口中，双击您要使用的会话的图标。
                                                      当您的设备连接到联机服务上后，将会出现“终端窗口”。
                                                    • 可以为会话创建桌面快捷方式，并通过双击进行连接。
                                                    • 若要断开连接，请选择“文件”>“取消”。

                                                    调整会话属性
                                                    1. 在“终端”窗口中，选择要修改的会话的图标。
                                                    2. 选择“属性”按钮。
                                                    3. 在“通信”选项卡上，根据需要修改设置。
                                                    4. 若要修改仿真设置，请选择“仿真”选项卡。
                                                    5. 在“选择仿真类型”列表中，选择要仿真的终端的类型。
                                                    6. 在“选择代码页”列表中，选择所需字符集的代码页选项。
                                                    7. 若要在发送前显示输入的文本，请选择“本地回显”。
                                                    8. 在“默认使用小字体”框中，设置字体首选项。
                                                    9. 在“回车->回车/换行”框中，调整回车/换行符，并选择“入站”或“出站”设置。
                                                    10. 在“自动滚动”框中，设置滚动首选项。
                                                    11. 完成后，选择“确定”。

                                                    注意：若要清除缓冲区，请选择“编辑”>“清除缓冲区”。
