      Sobre conexão com a Internet

        Você pode usar o dispositivo móvel para se conectar à Internet, o que permite navegar na World Wide Web (WWW) e trocar emails.

          Se o dispositivo não estiver equipado com um modem interno, será necessário instalar uma placa de modem ou modem externo. Consulte as instruções do fabricante do modem para obter detalhes sobre a instalação. Após configurar o modem, crie uma conexão para um provedor de serviços de Internet (ISP).

            Criar uma conta em um provedor de serviços de Internet

              Observação: as contas PPTP não são suportadas no dispositivo.

                Configure uma conta de protocolo ponto a ponto (PPP) com um provedor de serviços de Internet e obtenha as seguintes informações:

                • Número do telefone de acesso
                • Nome de usuário
                • Senha

                Após estabelecer uma conta, crie uma nova conexão no dispositivo. Quando criar essa conexão, você deve estar apto a usar todas as configurações padrão de TCP/IP fornecidas no Assistente Fazer Nova Conexão. Se você não puder se conectar usando essas configurações padrão, consulte o provedor para obter informações específicas sobre TCP/IP e para obter os endereços DNS primário e secundário.

                  Criar uma nova conexão no dispositivo
                    1. Selecione Iniciar > Configurações > Conexões Dial-up e de Rede.
                    2. Toque ou clique duas vezes no ícone Fazer Nova Conexão.
                    3. Na caixa de diálogo Fazer Nova Conexão, digite um nome para a conexão.
                    4. Selecione Conexão de discagem.
                    5. Selecione o botão Avançar.
                    6. Selecione o modem desejado na lista suspensa.
                    7. Para digitar um novo número de telefone a cada conexão, selecione Configurar.
                      • Na caixa de diálogo Propriedades do Dispositivo, selecione a guia Configurações de Porta.
                      • Marque a opção Discagem Manual para poder inserir um número de telefone a cada conexão.
                      • Para personalizar as opções do telefone, selecione a guia Opções de Chamada e defina as preferências de opção.
                      • Selecione OK para aceitar as opções de configuração.
                    8. Selecione Configurações de TCP/IP.
                      • Na guia Geral, verifique se a opção Usar end. IP atribuído pelo servidor está marcada.
                      • Na guia Servidores de Nomes, verifique se a opção Usar end. atribuídos pelo servidor está marcada.
                      • Selecione OK.
                    9. Selecione o botão Avançar.
                    10. Na caixa de diálogo Número do Telefone, insira o número do telefone de acesso ao provedor.
                    11. Selecione o botão Concluir.

                    A conexão criada aparece como um ícone no painel Conexões de Rede.

                    Observações

                    • Se você não puder se conectar com as configurações padrão, consulte seu provedor de serviços de Internet ou o administrador de rede para obter ajuda.
                    • Se você estiver criando uma conexão de modem de PC para um modem externo, selecione Hayes Compatible em COM1 como seu modem.
                    • Ao conectar o dispositivo a um PC usando um modem externo, use o cabo de modem ou o adaptador fornecido pelo fabricante do modem.

                        Sobre conexão a uma rede

                          É possível conectar-se diretamente a uma rede por uma conexão de rede (Ethernet) ou dial-up. Você pode acessar seus emails ou arquivos disponíveis no servidor de rede e navegar na Internet ou intranet. Para receber emails, crie um serviço de correio POP3 ou IMAP4 na Caixa de Entrada depois de criar a conexão com a rede. Para obter mais informações, consulte a Ajuda da Caixa de Entrada.

                            Criar uma conexão por rede (Ethernet)

                              1. No dispositivo, insira a placa Ethernet.
                              2. Na caixa de diálogo Configurações do Ethernet Driver, selecione Obter um endereço IP via DHCP. Selecione OK.
                              3. Selecione Iniciar > Configurações > Painel de Controle. Toque ou clique duas vezes no ícone Conexões Dial-up e de Rede.
                              4. Clique duas vezes no ícone da conexão para o adaptador e clique em OK para verificar as configurações.
                                Por exemplo, se você tiver um adaptador DC21x4 Ethernet, clique duas vezes no ícone de conexão DC211401.
                              5. Quando solicitado, digite o Nome de Usuário, Senha e Domínio usados para fazer logon na rede.

                                Criar uma conexão por rede dial-up

                                1. Peça ao administrador da rede para criar uma conta RAS para você. Assim que a conta for configurada, obtenha as seguintes informações: número de telefone dial-up, senha e nome do computador.
                                2. Crie uma nova conexão no dispositivo usando as informações da etapa 1. Você deverá usar as configurações padrão de TCP/IP fornecidas no Assistente Fazer Nova Conexão. Se você não puder se conectar usando essas configurações padrão, consulte o administrador de rede para obter informações específicas sobre TCP/IP.

                                Criar uma nova conexão no dispositivo

                                  Você deve criar uma conexão para cada computador ou rede que desejar acessar remotamente, a menos que esteja usando uma placa Ethernet.

                                  1. Selecione Iniciar > Configurações > Conexões Dial-up e de Rede.
                                  2. Toque ou clique duas vezes no ícone Fazer Nova Conexão.
                                  3. Na caixa de diálogo Fazer Nova Conexão, digite um nome para a conexão.
                                  4. Selecione Conexão Dial-up e clique no botão Avançar.
                                  5. Selecione o modem que deseja usar.
                                  6. Para digitar um novo número de telefone a cada conexão:
                                    • Selecione Discagem Manual e clique em OK.
                                    • Selecione Configurações de TCP/IP.
                                    • Na guia Geral, verifique se a opção Usar end. IP atribuído pelo servidor está marcada.
                                    • Na guia Servidores de Nomes, verifique se a opção Usar end. atribuídos pelo servidor está marcada e clique em OK.
                                    • Selecione o botão Avançar e digite o número de telefone.
                                    • Selecione o botão Concluir.
                                  A conexão que você criou aparece como um ícone da pasta Rede Remota.
                                    Observações
                                    • Se você não puder se conectar com essas configurações padrão, consulte seu provedor ou o administrador de rede para obter as configurações específicas de TCP/IP.
                                    • A conexão aparece como um ícone na tela Conexões.
                                    • Se você estiver criando uma conexão de modem de PC para um modem externo, selecione Hayes Compatible em COM1 como seu modem. Quando conectar o dispositivo ao PC usando um modem externo, use o cabo de modem ou o adaptador fornecido pelo fabricante do modem.

                                      Conectar-se a uma rede sem fio
                                      1. Insira sua placa de rede sem fio. Se solicitado, forneça o nome do driver fornecido pelo fabricante.
                                      2. Quando carregado, o sistema exibirá uma caixa de diálogo para ajudá-lo a se conectar a redes sem fio disponíveis.
                                      3. Selecione o nome da rede com a qual deseja se conectar e clique em Conectar.
                                      4. Confirme se as configurações foram detectadas e clique em OK.
                                      5. Se desejar revisar ou alterar as configurações, selecione Iniciar > Configurações > Conexões Dial-up e de Rede.
                                      6. Clique duas vezes no ícone de conexão remota que você deseja usar.
                                      7. Se houver suporte para o driver de sua placa de rede sem fio, a guia Redes sem Fio estará disponível para o ajuste de configurações.
                                      8. Use esta guia para exibir, alterar ou ajustar configurações.

                                        Modificar configurações da conexão
                                        1. Selecione Iniciar > Configurações > Conexões Dial-up e de Rede.
                                        2. Selecione o ícone para as configurações da conexão que você deseja modificar.
                                        3. Selecione Arquivo > Propriedades ou clique duas vezes no ícone apropriado.
                                        4. Selecione as opções desejadas. Pode haver configurações adicionais que dependam da conexão. Para modificá-las, selecione o ícone e selecione Configurações Avançadas... do menu.

                                        Desativar a compactação de software
                                        1. Selecione Iniciar > Configurações > Conexões Dial-up e de Rede.
                                        2. Selecione o ícone para a conexão cujas configurações você deseja modificar.
                                        3. Selecione Arquivo > Propriedades.
                                        4. Selecione Configurações de TCP/IP.
                                        5. Na guia Geral, limpe Usar compactação de software.

                                        Observação Ao mesmo tempo que a desativação da compactação de software reduz o tempo de conclusão de uma conexão, ela também reduz a taxa de transmissão de dados. Desative a compactação de software apenas se você se conectar freqüentemente e transferir pequenas quantidades de dados em cada sessão.

                                              Sobre o terminal

                                                Você pode conectar o dispositivo a um serviço online ou a um servidor corporativo que exija emulação de terminal TTY ou VT-100. Por exemplo, é possível exibir e baixar arquivos de um BBS ou enviar e receber email.

                                                  Use o terminal apenas quando se comunicar com serviços que exijam emulação de terminal.

                                                    Criar uma nova sessão de terminal
                                                    1. Selecione Iniciar > Configurações > Painel de Controle > Licenças do Cliente do Terminal Server.
                                                    2. Toque ou clique duas vezes no ícone Criar uma Nova Sessão.
                                                    3. Na caixa Nome da sessão, digite um nome para a sessão.
                                                    4. Na lista Selecionar um modem, selecione o nome de seu modem.
                                                    5. Digite o número de telefone para o computador remoto.
                                                    6. A menos que você precise ajustar as propriedades da emulação, selecione OK para se conectar.
                                                    Observações
                                                    • A sessão criada aparece como um ícone na pasta Terminal.
                                                    • Para desconectar, selecione Arquivo > Cancelar.

                                                    Ajustar propriedades da sessão
                                                    1. Na janela Terminal, selecione o ícone para a sessão que você deseja modificar.
                                                    2. Selecione o botão Propriedades.
                                                    3. Na guia Comunicações, modifique as configurações conforme necessário.
                                                    4. Para modificar as configurações de emulação, selecione a guia Emulação.
                                                    5. Na lista Escolher um tipo de emulação, selecione o tipo de terminal a ser emulado.
                                                    6. Na lista Seleção de página de código, selecione a opção de página de código para o conjunto de caracteres desejado.
                                                    7. Para exibir o texto digitado antes de enviar, selecione Eco local.
                                                    8. Na caixa Usar fonte pequena por padrão, defina a preferência de fonte.
                                                    9. Na caixa RC -> RC/AL, ajuste o retorno do carro/alimentação de linha e selecione as configurações Entrada ou Saída.
                                                    10. Na caixa Rolagem automática, defina a preferência de rolagem.
                                                    11. Quando terminar, selecione OK.

                                                    Observação Para limpar o buffer, selecione Editar > Limpar buffer.
