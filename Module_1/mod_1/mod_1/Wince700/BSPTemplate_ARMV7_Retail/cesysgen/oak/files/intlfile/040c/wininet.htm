Vue d'ensemble des certificats

Un certificat est un reçu numérique contenant des informations sur l'appareil qui présente ce reçu. Par exemple, un appareil tel qu'un serveur Web peut avoir un certificat contenant des informations qui le concernent. Les certificats valides sont signés par une autorité de certification (AC) et comprennent des informations sur la personne qui a autorisé le certificat, ses clés publiques et sa date d'expiration.

Lorsque vous êtes connecté à Internet, il est important que vous fassiez confiance à la certification des sites Web que vous visitez. Les sites Web qui ne sont pas approuvés ou correctement certifiés peuvent endommager les informations stockées sur votre appareil. Si vous avez un doute, vous pouvez afficher le certificat du serveur Web à partir de votre appareil et décider si vous souhaitez visiter le site Web.

Les options de sécurité et de confidentialité de votre appareil permettent de protéger votre vie privée et de sécuriser vos informations personnelles. Ces options de confidentialité vous permettent de protéger vos informations d'identification personnelles. Elles vous permettent de déterminer si les sites Web que vous visitez peuvent utiliser vos informations d'identification personnelles. Les contrôles de confidentialité de votre appareil vous permettent également de spécifier les paramètres de confidentialité afin de déterminer les informations à partager et à enregistrer lorsque vous visitez un site Web.

Certificats clients

Un certificat client est une instruction numérique vérifiant l'identité d'un utilisateur ou d'un appareil, tel qu'un ordinateur. Les certificats clients sont émis pour établir l'identité et pour fournir un niveau d'authentification et de sécurité aux informations échangées lors de l'utilisation de plusieurs appareils. Certains sites Web vous demandent de vous identifier à l'aide des certificats clients. Lorsque vous envoyez un certificat client à un autre ordinateur, vous utilisez des clés publiques et privées qui vous permettent de partager des données avec cet appareil de manière sécurisée. Les clés publiques et privées vous aident également à empêcher les utilisateurs non autorisés d'accéder à votre appareil et aux informations qu'il contient.

La liste ci-dessous décrit les informations contenues dans les certificats :

  • Valeur de la clé publique du sujet et informations sur l'identificateur
  • Informations sur l'identificateur de l'émetteur
  • Période de validité
  • Signature numérique de l'émetteur

Le sujet est l'entité qui reçoit le certificat. L'émetteur du certificat est l'autorité de certification.

Certificats racines et personnels

Vous pouvez stocker deux types de certificats sur votre appareil : des certificats personnels qui établissent votre identité et des certificats racines qui établissent l'identité des serveurs Web. Par exemple, lorsque vous envoyez des informations personnelles via Internet, un certificat personnel vous authentifie au site Web que vous visitez, alors qu'un certificat racine vous aide à protéger vos informations personnelles lorsque vous visitez le site.

Vue d'ensemble de la confidentialité

Les options de confidentialité de votre appareil vous permettent de protéger vos informations d'identification personnelles. Elles vous permettent de déterminer si les sites Web que vous visitez utilisent ces informations. Elles vous permettent également de spécifier les paramètres de confidentialité pour décider si vous souhaitez autoriser les sites Web à enregistrer des informations, telles que des cookies, sur votre ordinateur.

Cookies

Chaque fois que vous visitez un site Web, ce dernier stocke un petit fichier appelé « cookie » sur votre ordinateur. Les cookies sont créés par le site Web pour stocker des informations, telles que vos préférences lorsque vous visitez le site. Certains cookies contiennent un enregistrement des pages Web du site que vous avez visitées afin de permettre au site Web de personnaliser l'affichage lors de votre prochaine visite.

Les cookies peuvent également stocker vos informations d'identification personnelles, telles que votre nom, votre adresse électronique, l'adresse de votre travail ou de votre domicile ou votre numéro de téléphone. En revanche, un site Web a uniquement accès aux informations d'identification personnelles que vous fournissez.

La liste ci-dessous décrit les options de confidentialité de votre appareil :

  • Paramètres de confidentialité spécifiant la façon dont votre ordinateur gère les cookies
  • Alertes de confidentialité vous indiquant lorsque vous essayez d'accéder à un site qui ne répond pas aux critères définis dans vos paramètres de confidentialité
  • Possibilité d'afficher une politique en matière de confidentialité P3P du site Web

Vous pouvez utiliser les options de confidentialité de votre appareil pour limiter l'utilisation des cookies. En revanche, étant donné que de nombreuses applications dépendent des informations contenues dans les cookies, la désactivation de tous les cookies n'est pas toujours la meilleure solution.

Contenu interne et contenu tiers

Le contenu interne contient des informations associées au domaine hôte ou au site Web que vous visitez. Le contenu tiers provient de tout autre domaine. Par exemple, lorsque vous visitez un site Web contenant une bannière publicitaire, le site Web que vous visitez est considéré comme le contenu interne et la bannière publicitaire est considérée comme le contenu tiers.

De nombreuses pages Web utilisent à la fois des cookies internes et tiers pour obtenir et relayer les données correspondant à ces pages. Les options de confidentialité de votre appareil fait la distinction entre le contenu interne et le contenu tiers et établit une relation différente entre les deux. En réalité, les paramètres de confidentialité par défaut du contenu tiers sont plus rigoureux que du ceux contenu interne. Les paramètres de confidentialité de votre appareil Windows Embedded Compact vous permettent d'interdire les cookies tiers ainsi que d'autoriser les cookies internes ou de vous inviter à les autoriser ou à les interdire.

Vue d'ensemble de la sécurité

Les options de sécurité de votre appareil permettent d'empêcher les utilisateurs d'accéder aux informations auxquelles ils n'ont pas le droit d'accès, comme par exemple les informations sur votre carte de crédit que vous entrez lorsque vous faites des achats sur Internet. Elles vous permettent également de protéger votre ordinateur ou votre appareil mobile contre tout logiciel présentant des risques.

Vous pouvez parcourir de manière sécurisée un site Web qui utilise le protocole HTTPS (Hypertext Transfer Protocol over Secure Socket Layer). Le protocole HTTPS est un protocole de communications Web sécurisé qui crypte les données vers toute connexion utilisant HTTPS. Les données cryptées utilisant le protocole HTTPS ne peuvent pas être interceptées ni comprises par une tierce partie.

Alertes de sécurité

Les alertes de sécurité peuvent s'afficher pour les raisons suivantes :

  • Le serveur Web auquel vous vous connectez contient un certificat de serveur incorrect.
  • Le serveur Web auquel vous vous connectez contient une sécurité mixte (données sécurisées et non sécurisées).
  • Vous quittez une connexion sécurisée.
