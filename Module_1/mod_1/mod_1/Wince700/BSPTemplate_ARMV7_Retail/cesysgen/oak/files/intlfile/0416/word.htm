    Sobre o WordPad

      É possível criar e editar documentos e modelos no WordPad como no Microsoft Word usando botões e comandos de menu semelhantes aos usados na versão do Microsoft Word para computadores de mesa. Você pode trabalhar com arquivos no modo Normal ou em Tópicos.

        Os documentos do WordPad são normalmente gravados como arquivos .pwd, mas é possível salvá-los em outros formatos, como .rtf ou .doc.

            Selecionando texto
              Para selecionar Faça isto
              Qualquer quantidade de texto Arraste a caneta ou o ponteiro sobre o texto.
              Uma palavra Clique duas vezes na palavra.
              Um único parágrafo Clique três vezes no parágrafo.
              Um documento inteiro Selecione Editar > Selecionar tudo.
              Uma figura Toque ou clique na figura.


              Localizando texto
              1. Para pesquisar por texto em uma parte de um documento, selecione essa parte.
                Caso contrário, o documento todo será pesquisado.
              2. Selecione Editar > Localizar.
              3. Digite o texto que você deseja pesquisar.
                Para um parágrafo, digite ^p. Para uma parada de tabulação, digite ^t.
              4. Selecione as opções de pesquisa desejadas.
              5. Selecione Localizar próxima.
              6. Para continuar pesquisando, selecione Editar > Localizar próxima.

              Substituir texto

              Substituir texto
              1. Para substituir texto em uma parte do documento, selecione essa parte.
                Caso contrário, o documento todo será pesquisado.
              2. Selecione Editar > Substituir.
              3. Digite o texto que deseja localizar e o texto pelo qual deseja substituí-lo.
                Para um parágrafo, digite ^p. Para uma parada de tabulação, digite ^t.
              4. Selecione as opções de substituição desejadas.
              5. Selecione Localizar próxima.
              6. Na barra de ferramentas Localizar/Substituir, selecione o botão apropriado.

              Movendo, copiando ou excluindo texto
              1. Selecione o texto a ser movido, copiado ou excluído.
              2. Para mover ou excluir o texto, selecione o botão Recortar.
                Para copiar o texto, selecione o botão Copiar.
              3. Para colar as informações, mova o ponto de inserção para o local desejado e selecione o botão Colar.
              Observações
              • Para desfazer uma ação, selecione Editar > Desfazer.
              • Para restaurar uma ação, selecione Editar > Refazer.

                Criando listas numeradas e de marcadores
                1. Selecione o botão Marcadores ou Numeração.
                2. Digite o primeiro item.
                3. Pressione ENTER, digite o próximo item e repita como necessário.
                4. Selecione o botão novamente para terminar a formatação.

                Observação Para alterar o estilo do número da lista selecionada, selecione Formatar > Parágrafo. Na lista Formatar, selecione o estilo desejado.

                  Configurando a fonte padrão
                  1. Selecione Formatar > Fonte.
                  2. Selecione a fonte desejada.
                  3. Selecione Configurar como padrão.
                  4. Para confirmar que as alterações serão aplicadas aos próximos documentos que você criar, selecione Sim.
                  Observação Para aplicar a fonte padrão e o estilo ao texto selecionado, selecione Aplicar padrão.

                            Inserindo símbolos
                            1. Coloque o ponto de inserção onde você deseja inserir um símbolo.
                            2. Selecione Ferramentas > Inserir símbolos.
                            3. Selecione o símbolo desejado e, em seguida, selecione Inserir.
                            Dicas
                            • Se você selecionar uma fonte diferente (ou um subconjunto de uma fonte) um conjunto de símbolos diferente será exibido.

                            Imprimindo um documento
                            1. Selecione Arquivo > Imprimir.
                            2. Selecione as opções de impressão desejadas.
                            Dicas
                            • Porta lista as portas de impressora disponíveis.
                            • Se Rede estiver selecionada na lista Porta, insira o caminho da impressora de rede na caixa Caminho de rede.

                            Configurando uma senha para um documento
                            1. Selecione Arquivo > Senha.
                            2. Digite e verifique a senha.
                            3. Você deve salvar o documento (Arquivo > Salvar) para reter a senha.

                              Sobre a conversão de documentos

                                Se um programa de correio eletrônico estiver instalado no dispositivo e você receber um documento ou um modelo do Word anexado, criado no Word versão 6.0 ou superior, o WordPad converterá o arquivo no dispositivo. Para exibir o documento, simplesmente abra o anexo no programa de correio eletrônico e abra o arquivo no WordPad.

                                  Durante a conversão, alguns atributos de formatação talvez sejam alterados ou perdidos. Para evitar a perda permanente desses atributos, feche o arquivo depois de exibi-lo invés de salvá-lo no dispositivo. Se você salvar o arquivo, salve-o com um nome diferente.

                                    Para mais informações sobre conversões executadas no dispositivo, consulte os tópicos a seguir.

                                        Formatação

                                          Bordas e sombreamento
                                          Bordas e sombreamento não são exibidos no WordPad, mas são restaurados quando convertidos de volta para um documento do Word. O sombreamento não é restaurado quando utilizado em uma tabela, mas bordas simples são restauradas.

                                            Formatação de caracteres
                                            Negrito, itálico, tachado, sobrescrito, subscrito e texto oculto são mantidos e exibidos. Outros efeitos são alterados ou removidos.

                                              Cores
                                              As cores são mantidas e mapeadas para as cores disponíveis no dispositivo.

                                                Fontes e tamanhos de fonte
                                                As fontes que não são suportadas pelo dispositivo móvel são mapeaas para a fonte mais próxima disponível no WordPad.

                                                  Formatação da página
                                                  Cabeçalhos, rodapés, notas de rodapé, colunas, informações de configuração da página e folhas de estilo são removidos durante a conversão para o WordPad. O WordPad suporta cabeçalhos predefinidos.

                                                    Formatação do parágrafo
                                                    Tabulações, alinhamento, marcadores, recuos, listas de numeração simples e espaçamento entre parágrafos são mantidos e exibidos no WordPad.


                                                      Figuras

                                                        As figuras são removidas durante a conversão.


                                                        Índice analítico

                                                          O texto do índice analítico e algumas formatações são preservados. Entretanto, todo texto configurado com paradas de tabulações alinhadas à direita talvez passem para a próxima linha.


                                                          Índice

                                                            O texto do índice e algums formatações são preservados.


                                                            Tabelas

                                                              As tabelas aparecem como texto delimitado por tabulações. As células contendo texto ajustado, tabulações ou marcas de parágrafo talvez apresentem dificuldade de leitura. Tabelas simples são exibidas de forma precisa e restauradas quando o arquivo do WordPad é convertido de volta para um documento do Word.


                                                              Objetos OLE

                                                                Os objetos OLE são removidos durante a conversão.


                                                                Marcas de revisão, anotações e comentários

                                                                  As marcas de revisão são visíveis mas não podem ser alteradas. As anotações e os comentários são removidos durante a conversão.
