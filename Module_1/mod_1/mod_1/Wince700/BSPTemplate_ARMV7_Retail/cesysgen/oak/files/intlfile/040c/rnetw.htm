      À propos de la connexion à Internet

        Vous pouvez utiliser votre appareil mobile pour vous connecter à Internet, ce qui vous permet de naviguer sur le Web et d'échanger du courrier électronique.

          Si votre appareil n'est pas équipé d'un modem incorporé, vous devrez installer une carte modem ou un modem externe. Pour plus de détails sur l'installation, suivez les instructions du fabricant de votre modem. Dès que le modem est installé, créez une connexion à un FAI.

            Création d'un compte auprès d'un FAI

              Remarque : les comptes PPTP ne sont pas pris en charge par votre appareil.

                Établissez un compte PPP (Point-to-Point Protocol) avec un FAI (Fournisseur d'accès à Internet) et obtenez les informations suivantes :

                • Numéro de téléphone d'accès
                • Nom d'utilisateur
                • Mot de passe

                Dès que vous avez établi un compte, créez une nouvelle connexion sur votre appareil. Lors de la création de cette connexion, vous devriez pouvoir utiliser tous les paramètres TCP/IP par défaut proposés dans l'Assistant Établir une nouvelle connexion. Si vous ne pouvez pas vous connecter en utilisant ces paramètres par défaut, contactez votre FAI pour obtenir des informations TCP/IP précises, ainsi que les adresses DNS principale et secondaire.

                  Création d'une nouvelle connexion sur votre appareil
                    1. Sélectionnez Démarrer > Paramètres > Connexions réseau et accès à distance.
                    2. Appuyez deux fois ou double-cliquez sur l'icône Établir une nouvelle connexion.
                    3. Dans la boîte de dialogue Établir une nouvelle connexion, entrez le nom de la connexion.
                    4. Sélectionnez Connexion à distance.
                    5. Cliquez sur Suivant.
                    6. Sélectionnez le modem que vous voulez utiliser dans la liste déroulante.
                    7. Si vous voulez composer un nouveau numéro de téléphone à chaque connexion, sélectionnez Configurer.
                      • Dans la boîte de dialogue Propriétés de l'appareil, cliquez sur l'onglet Paramètres des ports.
                      • Activez la case à cocher Numérotation manuelle pour pouvoir composer un numéro de téléphone à chaque connexion.
                      • Pour personnaliser les options téléphoniques, cliquez sur l'onglet Options d'appel et définissez vos préférences.
                      • Cliquez sur OK pour confirmer vos choix de configuration.
                    8. Sélectionnez Paramètres TCP/IP.
                      • Dans l'onglet Général, assurez-vous que l'option Utiliser l'adresse IP attribuée par le serveur est activée.
                      • Dans l'onglet Serveurs de noms, assurez-vous que l'option Utiliser les adresses attribuées par le serveur est activée.
                      • Sélectionnez OK.
                    9. Cliquez sur Suivant.
                    10. Dans la boîte de dialogue Numéro de téléphone, tapez le numéro de téléphone de votre FAI.
                    11. Sélectionnez le bouton Terminer.

                    La connexion que vous venez de créer apparaît sous la forme d'une icône dans le panneau Connexions réseau.

                    Remarques

                    • S'il vous est impossible de vous connecter en utilisant ces paramètres par défaut, consultez votre fournisseur d'accès Internet ou votre administrateur réseau pour obtenir de l'aide.
                    • Si vous établissez une connexion par modem de PC pour un modem externe, sélectionnez Hayes Compatible sur COM1 comme modem.
                    • Lorsque vous connectez l'appareil à un PC à l'aide d'un modem externe, utilisez le câble ou l'adaptateur modem fourni par le fabricant de votre modem.

                        À propos de la connexion à un réseau

                          Vous pouvez vous connecter directement à un réseau à l'aide d'une connexion réseau (Ethernet) ou d'une connexion à distance. Vous pouvez accéder à votre courrier électronique, accéder à des fichiers disponibles sur le serveur du réseau et naviguer sur Internet ou un intranet. Pour recevoir votre courrier électronique, créez un service de messagerie POP3 ou IMAP4 dans la Boîte de réception après avoir établi la connexion au réseau. Pour plus d'informations, consultez l'Aide de la Boîte de réception.

                            Création d'une connexion réseau (Ethernet)

                              1. Insérez la carte Ethernet dans votre appareil.
                              2. Dans la boîte de dialogue Paramètres de pilote Ethernet, sélectionnez Obtenir une adresse IP par DHCP. Sélectionnez OK.
                              3. Sélectionnez Démarrer > Paramètres > Panneau de configuration. Appuyez deux fois ou double-cliquez sur l'icône Connexions réseau et accès à distance.
                              4. Double-cliquez sur l'icône de connexion pour la carte, et cliquez sur OK pour vérifier les paramètres.
                                Par exemple, si vous avez une carte Ethernet DC21x4, double-cliquez sur l'icône de connexion DC211401.
                              5. À l'invite, entrez le Nom d'utilisateur, le Mot de passe et le nom de Domaine que vous utilisez pour vous connecter au réseau.

                                Création d'une connexion à distance

                                1. Demandez à l'administrateur réseau de vous créer un compte RAS. Dès que le compte est défini, recherchez les informations suivantes : numéro de téléphone, mot de passe et nom de l'ordinateur.
                                2. Établissez une nouvelle connexion sur votre appareil en utilisant les informations de l'étape 1. Vous devriez pouvoir utiliser les paramètres TCP/IP par défaut dans l'Assistant Établir une nouvelle connexion. Si vous ne pouvez pas vous connecter en utilisant ces paramètres par défaut, contactez votre administrateur réseau pour obtenir des informations TCP/IP précises.

                                Création d'une nouvelle connexion sur votre appareil

                                  Vous devez établir une nouvelle connexion pour chaque ordinateur ou réseau auquel vous souhaitez accéder, sauf si vous utilisez une carte Ethernet.

                                  1. Sélectionnez Démarrer > Paramètres > Connexions réseau et accès à distance.
                                  2. Appuyez deux fois ou double-cliquez sur l'icône Établir une nouvelle connexion.
                                  3. Dans la boîte de dialogue Établir une nouvelle connexion, entrez le nom de la connexion.
                                  4. Sélectionnez Connexion à distance et cliquez sur le bouton Suivant.
                                  5. Sélectionnez le modem que vous souhaitez utiliser.
                                  6. Pour composer un nouveau numéro de téléphone à chaque connexion :
                                    • Sélectionnez Appel manuel, puis cliquez sur OK.
                                    • Sélectionnez Paramètres TCP/IP.
                                    • Dans l'onglet Général, assurez-vous que l'option Utiliser l'adresse IP attribuée par le serveur est activée.
                                    • Dans l'onglet Serveurs de noms, assurez-vous que l'option Utiliser les adresses attribuées par le serveur est activée et choisissez OK.
                                    • Sélectionnez le bouton Suivant, puis tapez le numéro de téléphone.
                                    • Sélectionnez le bouton Terminer.
                                  La connexion que vous venez de créer apparaît sous la forme d'une icône dans le dossier Réseau.
                                    Remarques
                                    • S'il vous est impossible de vous connecter en utilisant ces paramètres par défaut, consultez votre fournisseur d'accès Internet ou votre administrateur réseau pour obtenir des informations TCP/IP ou de configuration précises.
                                    • La connexion apparaît sous la forme d'une icône sur l'écran Connexions.
                                    • Si vous établissez une connexion par modem de PC pour un modem externe, sélectionnez Hayes Compatible sur COM1 comme modem. Lorsque vous connectez l'appareil au PC à l'aide d'un modem externe, utilisez le câble ou l'adaptateur modem fourni par le fabricant de votre modem.

                                    Connexion à un réseau

                                    Si vous vous connectez à un réseau, vous pourrez uniquement recevoir des messages électroniques et naviguer sur Internet. Pour recevoir des messages électroniques, créez un service de messagerie POP3 ou IMAP4 dans la Boîte de réception. Pour plus d'informations, consultez l'Aide de la Boîte de réception.

                                      1. Sélectionnez Démarrer > Paramètres > Connexions réseau et accès à distance.
                                      2. Appuyez deux fois ou double-cliquez sur l'icône de connexion à distance à utiliser.

                                        La première fois que vous vous connectez, vous devrez entrer votre nom d'utilisateur, votre mot de passe et votre domaine. Pour éviter de devoir retaper votre mot de passe lors des connexions ultérieures, sélectionnez Enregistrer le mot de passe.
                                      3. Sélectionnez Se connecter.
                                      4. Pour vous déconnecter, sélectionnez Déconnecter.

                                      Connexion à un réseau sans fil
                                      1. Branchez votre carte sans fil. À l'invite, indiquez le nom du pilote fourni par le fabricant.
                                      2. Au terme du chargement, le système affichera une boîte de dialogue qui vous aidera à vous connecter aux réseaux sans fil disponibles.
                                      3. Sélectionnez le nom du réseau auquel vous souhaitez vous connecter et cliquez sur Connecter.
                                      4. Confirmez les paramètres qui ont été détectés et cliquez sur OK.
                                      5. Pour vérifier ou modifier des paramètres, sélectionnez Démarrer > Paramètres > Connexions réseau et accès à distance.
                                      6. Double-cliquez sur l'icône de connexion à distance que vous souhaitez utiliser.
                                      7. Si le pilote de votre carte sans fil le prend en charge, un onglet Réseaux sans fil vous permettra de régler les paramètres.
                                      8. Cet onglet vous permet de visualiser, de modifier ou de régler des paramètres.

                                        Modification des paramètres de connexion
                                        1. Sélectionnez Démarrer > Paramètres > Connexions réseau et accès à distance.
                                        2. Sélectionnez l'icône de la connexion dont vous souhaitez modifier les paramètres.
                                        3. Sélectionnez Fichier > Propriétés ou double-cliquez sur l'icône appropriée.
                                        4. Sélectionnez les options souhaitées. Il se peut qu'il y ait d'autres paramètres en fonction de la connexion. Pour les modifier, sélectionnez l'icône, puis Paramètres avancés dans le menu.
                                        Voir aussi

                                        Désactivation de la compression logicielle
                                        1. Sélectionnez Démarrer > Paramètres > Connexions réseau et accès à distance.
                                        2. Sélectionnez l'icône de la connexion dont vous souhaitez modifier les paramètres.
                                        3. Sélectionnez Fichier > Propriétés.
                                        4. Sélectionnez Paramètres TCP/IP.
                                        5. Dans l'onglet Général, désactivez la case à cocher Utiliser la compression logicielle.

                                        Remarque : la désactivation de la compression logicielle réduit la durée de la connexion, mais ralentit le taux de transmission des données. Ne désactivez la compression logicielle que si vous vous connectez fréquemment et que vous ne transférez que de petites quantités de données lors de chaque session.

                                            Important La modification de ce paramètre risque d'entraver la communication avec votre PC. Le débit en bauds que vous avez sélectionné pour votre appareil doit correspondre à celui de votre PC. Pour plus d'informations sur la modification du débit en bauds sur votre PC, voir l'Aide sur les Services Windows Embedded Compact.

                                            1. Sélectionnez Démarrer > Paramètres > Panneau de configuration.
                                            2. Appuyez deux fois ou double-cliquez sur l'icône Connexions PC.
                                            3. Sélectionnez Changer.
                                            4. Sélectionnez le débit en bauds souhaité.

                                              À propos du terminal

                                                Vous pouvez connecter votre appareil à un service en ligne ou à un serveur d'entreprise qui requiert une émulation de terminal TTY ou VT-100. Par exemple, vous pouvez afficher et télécharger des fichiers à partir d'une information télématique ou envoyer et recevoir du courrier électronique.

                                                  N'utilisez le terminal que lorsque vous êtes en communication avec des services qui requièrent une émulation de terminal.

                                                    Création d'une nouvelle session de terminal
                                                    1. Sélectionnez Démarrer > Paramètres > Panneau de configuration > Licences Client Terminal Server.
                                                    2. Appuyez deux fois ou double-cliquez sur l'icône Créer une nouvelle session.
                                                    3. Dans la zone Nom de session, entrez le nom de la session.
                                                    4. Dans la liste Sélectionner un modem, sélectionnez le nom de votre modem.
                                                    5. Entrez le numéro de téléphone de l'ordinateur distant.
                                                    6. Sélectionnez OK pour vous connecter, sauf si vous devez régler les propriétés de l'émulation.
                                                    Remarques
                                                    • La session que vous avez créée apparaît sous la forme d'une icône dans le dossier Terminal.
                                                    • Pour vous déconnecter, sélectionnez Fichier > Annuler.

                                                      Lorsque votre appareil est connecté au service en ligne, la fenêtre Terminal apparaît.
                                                    • Vous pouvez créer un raccourci sur le bureau pour la session, et vous connecter en tapant deux fois dessus.
                                                    • Pour vous déconnecter, sélectionnez Fichier > Annuler.

                                                    Réglage des propriétés de session

                                                    Réglage des propriétés de session
                                                    1. Dans la fenêtre Terminal, sélectionnez l'icône de la session que vous souhaitez modifier.
                                                    2. Sélectionnez le bouton Propriétés.
                                                    3. Dans l'onglet Communications, modifiez les paramètres comme vous le souhaitez.
                                                    4. Pour modifier les paramètres d'émulation, sélectionnez l'onglet Émulation.
                                                    5. Dans la liste Choisir un type d'émulation, sélectionnez le type de terminal à émuler.
                                                    6. Dans la liste Sélection de page de code, sélectionnez l'option de page de code pour le jeu de caractères que vous souhaitez.
                                                    7. Pour afficher le texte que vous tapez avant de l'envoyer, sélectionnez Écho local.
                                                    8. Dans la zone Utiliser une petite police par défaut, définissez vos préférences pour la police.
                                                    9. Dans la zone CR -> CR/LF, réglez le retour chariot/ saut de ligne et sélectionnez les paramètres Entrant ou Sortant.
                                                    10. Dans la zone Défilement automatique, définissez vos préférences pour le défilement.
                                                    11. Lorsque vous avez terminé, sélectionnez OK.

                                                    Remarque : pour effacer le tampon, sélectionnez Edition > Effacer tampon.
