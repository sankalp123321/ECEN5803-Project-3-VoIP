    프로그램 시작

      바탕 화면에 없는 프로그램을 열려면 시작 메뉴, 프로그램을 선택한 다음, 프로그램 이름을 선택합니다.


      상황에 맞는 도움말 사용
      • 프로그램이나 대화 상자에 대한 자세한 내용을 보려면 해당 프로그램이나 대화 상자를 사용하는 동안 ? 단추를 선택합니다.
      • 도구 모음 단추의 이름을 보려면 단추 위에서 스타일러스를 누른 채로 둡니다. 단추 명령이 실행되지 않게 하려면 단추에서 스타일러스를 뗄 때까지 계속 누르고 있어야 합니다.
      • 마우스를 사용하여 도구 모음 단추 이름을 보려면 단추 위로 마우스 포인터를 가져다 댑니다.

      상황에 맞는 메뉴 열기
      • 스타일러스를 사용하는 경우 Alt 키를 누른 다음 해당 항목을 눌러 상황에 맞는 메뉴를 엽니다.
      • 마우스를 사용하는 경우 마우스 오른쪽 단추로 해당 항목을 클릭한 다음 상황에 맞는 메뉴를 엽니다.

              메뉴와 도구 모음 단추 간 전환

                슬라이더를 사용하여 메뉴와 도구 모음 표시를 제어합니다.

                • 메뉴와 도구 모음 단추 사이를 빠르게 전환하려면 슬라이더를 선택합니다.
                • 슬라이더를 좌우로 끌어서 메뉴 또는 도구 모음 단추를 표시하거나 메뉴와 도구 모음 단추를 조합하여 함께 표시합니다.
                • 메뉴와 도구 모음 단추를 모두 표시하려면 슬라이더를 메뉴 아래로 끕니다.


                파일을 메모리 카드로 이동

                1. 이동할 파일을 엽니다.
                2. 파일 > 다른 이름으로 저장을 선택합니다.
                3. 메모리 카드 폴더에 파일을 저장합니다.

                Windows 탐색기 사용

                  Windows 탐색기는 이동 장치에서도 PC에서와 거의 동일하게 작동합니다. 장치에서 Windows 탐색기를 사용하는 경우의 차이점에 대한 자세한 내용은 아래 항목을 참조하십시오.

                    파일 보기로 탐색

                    바탕 화면 바로 가기 만들기
                    인터넷으로 전환

                    파일 보기로 탐색

                      앞뒤로 이동
                      이동 메뉴 사용

                      앞뒤로 이동
                      • 이전 폴더로 이동하려면 이동 > 뒤로를 선택합니다.
                      • 바로 전에 보았던 다음 폴더로 이동하려면 이동 > 앞으로를 선택합니다.

                      이동 메뉴 사용
                      • My Documents 폴더를 빠르게 액세스하려면 이동 > My Documents를 선택합니다.
                      • 액세스한 모든 폴더를 보려면 이동 > 열어 본 페이지 목록을 선택합니다. 폴더를 두 번 클릭하면 해당 폴더로 돌아갑니다.

                      파일을 아이콘 또는 목록으로 보기
                      1. 아이콘을 표시하려면 보기 > 큰 아이콘 또는 작은 아이콘을 선택합니다.
                      2. 목록을 표시하려면 보기 > 자세히를 선택합니다.

                      파일 및 폴더 작업
                        새 폴더 만들기
                        즐겨찾기 목록에 폴더 추가
                        적외선 통신을 사용하여 파일 전송

                        새 폴더 만들기
                        1. 필요한 경우 새 폴더를 위치시킬 폴더를 엽니다.
                        2. 파일 > 새 폴더를 선택합니다.

                        즐겨찾기 목록에 폴더 추가
                        1. 폴더를 엽니다.
                        2. 즐겨찾기 > 즐겨찾기에 추가를 선택합니다.
                        3. 이름 상자에 바로 가기 이름을 입력합니다.
                        팁
                        • 즐겨찾기 파일 또는 폴더를 열려면 즐겨찾기를 선택한 다음 해당 폴더를 선택합니다.
                        • 즐겨찾기를 구성하려면 즐겨찾기 > 즐겨찾기 구성을 선택합니다.

                        적외선 통신을 사용하여 파일 전송
                        • 파일을 보내려면 해당 파일을 선택하고, 적외선 포트에 연결한 다음 파일 > 보내기 > 적외선으로를 선택합니다.
                        • 파일을 받으려면 적외선 포트를 연결하고 파일 > 받기를 선택합니다.

                        바탕 화면 바로 가기 만들기
                        1. 바로 가기를 만들 파일 또는 웹 페이지를 표시합니다.
                        2. 파일 > 보내기를 선택합니다.
                        3. 바탕 화면 바로 가기를 선택합니다.

                          인터넷으로 전환
                          • 주소 표시줄에 URL을 입력합니다.

                            Pocket Internet Explorer 정보

                              Pocket Internet Explorer를 사용하여 장치에서 인터넷 또는 인트라넷 웹 페이지를 볼 수 있습니다. 인터넷 서비스 공급자(ISP) 또는 네트워크에 연결하려면 모뎀 또는 이더넷 카드를 사용해야 합니다.

                                  인터넷에 연결

                                    웹 페이지를 보려면 ISP 또는 네트워크에 대한 원격 연결을 설정해야 합니다. ISP 또는 네트워크에 연결을 설정하지 않은 경우 원격 연결을 참조하십시오.

                                    1. 시작 > 프로그램 > 통신 > 원격 네트워킹을 선택합니다. 사용할 연결을 두 번 클릭합니다.
                                    2. 필요한 사용자 정보를 입력합니다. 전화 거는 방법을 조정하려면 전화 걸기 등록 정보를 선택합니다.
                                    3. 연결을 선택합니다.

                                    참고 연결하는 데 문제가 있는 경우에는 프록시 서버를 사용하여 인터넷을 액세스해야 할 수 있습니다.

                                      참고 항목

                                      자동 인터넷 연결 설정

                                        자동 인터넷 연결을 설정하면 Pocket Internet Explorer가 인터넷 또는 회사 네트워크에 액세스해야 하는 경우 선택한 원격 연결에 자동으로 전화를 겁니다.

                                        1. 보기 > 옵션 > 자동 연결 탭을 선택합니다.
                                        2. 자동 연결 사용을 선택합니다.
                                        3. 사용할 원격 연결을 지정합니다.
                                        4. 특정 시간 동안 유휴 상태(장치가 사용되지 않는 시간)이면 인터넷 연결이 자동으로 해제되게 하려면 자동 연결 끊기를 선택합니다.
                                        5. 유휴 시간이 <시간>분이면 연결끊기에 인터넷 연결을 해제할 유휴 상태 지속 시간을 입력합니다.
                                        참고 항목
                                        인터넷에 연결

                                        프록시 서버 설정

                                          회사 네트워크와 같은 로컬 네트워크를 통해 인터넷에 연결하는 경우, 일반적으로 프록시 서버를 사용하여 보안을 강화합니다.

                                          1. 보기 > 옵션 > 프록시 서버 탭을 선택합니다.
                                          2. 프록시 서버 사용을 선택합니다.
                                          3. 프록시 서버 주소와 포트를 입력합니다. 자세한 내용은 네트워크 관리자를 참조하십시오.
                                          4. 회사 인트라넷 페이지와 같은 로컬 주소에 프록시 서버를 사용하지 않으려면 로컬 주소에 프록시 사용 안함을 선택합니다.

                                          쿠키 사용

                                            쿠키 파일에는 사용자의 신원과 기본 설정에 대한 정보가 들어 있습니다. 웹 사이트에서는 쿠키 파일을 이용하여 사용자의 필요에 맞게 정보를 제공할 수 있습니다. 웹 사이트에서 쿠키 파일을 보내면 이동 장치에 쿠키가 저장됩니다.

                                            1. 보기 > 옵션 > 고급 탭을 선택합니다.
                                            2. 쿠키 사용을 선택합니다.

                                                              Pocket Internet Explorer는 보안 사이트에서 사용되는 보안 프로토콜을 지원합니다. 보안 사이트를 액세스할 때 개인 정보를 안전하게 보낼 수 있습니다. 웹 사이트 보안 여부를 확인하려면 상태 표시줄의 Pocket Internet Explorer 단추에서 잠금 아이콘을 확인합니다.

                                                                2. 보안 설정을 선택합니다.
                                                                3. 원하는 보안 프로토콜을 선택합니다.
                                                                  • PCT는 Private Communications Technology의 약어이며 Microsoft가 개발한 특허 프로토콜로서 SSL2보다 훨씬 안전합니다. Windows Embedded Compact OS는 PCT를 지원하지 않습니다.
                                                                  • SSL2는 Secured Sockets Layer Level 2의 약어로서 보안 전송을 위한 표준 프로토콜입니다. 모든 보안 인터넷 사이트는 이 프로토콜을 지원합니다.

                                                                                                                                      정의한 사용자 정의 색을 추가합니다.

                                                                                                                                        색/단색
                                                                                                                                        상자 왼쪽에는 현재 색의 흰색과 검정색의 양이 표시됩니다. 상자 오른쪽에는 흰색과 검정색이 섞이지 않은 색의 100%가 표시됩니다. 오른쪽의 슬라이더를 사용하여 조정합니다.
                                                                                                                                        흰색 또는 검정색이 섞이지 않은 색의 100%를 선택하려면 Alt+O를 입력합니다.
                                                                                                                                        색을 변경하려면 아래 설명된 값을 변경합니다. 또는 색 매트릭스를 선택하고 끌어서 값을 변경합니다.

                                                                                                                                          색상
                                                                                                                                          색 구성표의 값입니다. 빨강 = 0, 노랑 = 60, 녹색 = 120, 청록 = 180, 자홍 = 200, 파랑 = 240입니다.

                                                                                                                                            채도
                                                                                                                                            채도 수준, 즉, 지정된 채도의 색의 양은 최대 240입니다.

                                                                                                                                              명도
                                                                                                                                              명도 또는 밝기입니다.명도를 변경하면 빨강, 파랑 및 녹색의 값이 변경됩니다.

                                                                                                                                                빨강
                                                                                                                                                선택된 색에 포함된 빨강의 양입니다.

                                                                                                                                                  녹색
                                                                                                                                                  선택된 색에 포함된 녹색의 양입니다.

                                                                                                                                                    파랑
                                                                                                                                                    선택된 색에 포함된 파랑의 양입니다.


                                                                                                                                                      글꼴 대화 상자

                                                                                                                                                        참고 사용 가능한 옵션은 사용하고 있는 프로그램에 따라 다릅니다. 일부 옵션은 개별 프로그램에 표시되지 않거나 사용할 수 없습니다.

                                                                                                                                                          글꼴
                                                                                                                                                          현재 글꼴이 표시됩니다. 글꼴을 변경하려면 화살표를 선택하고 목록에서 원하는 글꼴을 선택합니다.

                                                                                                                                                            크기
                                                                                                                                                            현재 크기가 표시됩니다. 크기를 변경하려면 화살표를 선택하고 목록에서 원하는 크기를 선택합니다.

                                                                                                                                                              글꼴 유형
                                                                                                                                                              텍스트를 굵게, 기울임꼴, 밑줄 등의 글꼴로 표시하거나 이들을 조합하여 표시합니다.

                                                                                                                                                                색
                                                                                                                                                                장치가 현재 색을 지원하는 경우 현재 색을 표시합니다. 색을 변경하려면 화살표를 선택하고 목록에서 원하는 색을 선택합니다.

                                                                                                                                                                  미리 보기
                                                                                                                                                                  선택 옵션으로 서식을 지정한 텍스트가 표시됩니다.

                                                                                                                                                                    기본값 적용
                                                                                                                                                                    변경 내용을 무시하고 기본 서식을 복원합니다.

                                                                                                                                                                      기본값으로 설정
                                                                                                                                                                      현재 설정을 기본 텍스트 서식으로 사용하여 새로 만드는 문서에 적용합니다.


                                                                                                                                                                        대화 상자 열기

                                                                                                                                                                          한 수준 위로
                                                                                                                                                                          폴더 구조에서 위로 이동합니다.

                                                                                                                                                                            새 폴더
                                                                                                                                                                            새 폴더를 만듭니다. 원하는 이름을 입력합니다.

                                                                                                                                                                              목록
                                                                                                                                                                              폴더 및 파일을 아이콘으로 표시합니다.

                                                                                                                                                                                자세히
                                                                                                                                                                                폴더와 파일을 목록에 표시하여 크기, 유형, 마지막으로 수정한 날짜 등의 파일 정보를 제공합니다.

                                                                                                                                                                                  이름
                                                                                                                                                                                  현재 선택된 폴더 또는 파일의 이름이 표시됩니다.

                                                                                                                                                                                    형식
                                                                                                                                                                                    표시할 파일 형식을 선택합니다. 현재 폴더에 있는 모든 파일을 표시하려면 모든 문서(*.*)를 선택합니다.


                                                                                                                                                                                      파일에 암호 설정
                                                                                                                                                                                      1. 파일 > 암호를 선택합니다.
                                                                                                                                                                                      2. 암호를 입력하고 확인합니다.
                                                                                                                                                                                      3. 파일을 저장(파일 > 저장)해야만 암호를 유지할 수 있습니다.

                                                                                                                                                                                      참고 암호로 보호된 파일은 동기화할 수 없습니다. PC 또는 장치에서 해당 프로그램의 암호 보호를 제거한 다음 동기화해야 합니다. 동기화되면 암호 보호를 다시 설정할 수 있습니다.
