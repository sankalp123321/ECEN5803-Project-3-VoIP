    关于 WordPad

      可以在 WordPad 中创建、编辑文档和模板，所使用的按钮和菜单命令与 Microsoft Word 桌面版本中的类似。可以在“普通”或“大纲”视图中处理文件。

        WordPad 文档通常保存为 .pwd 文件，但也可以保存为其他文件格式，如 .rtf 或 .doc。

            选择文本
              选择 方法
              任意数量的文本 将笔针和指针拖至文本上。
              一个字 双击该字。
              一个段落 在该段落内点击三下。
              整篇文档 选择“编辑”>“全选”。
              一幅图片 点击或单击该图片。


              查找文本
              1. 若要在文档的某部分中搜索文本，请选择此部分。
                否则，将在整篇文档中搜索。
              2. 选择“编辑”>“查找”。
              3. 输入要查找的文本。
                对于段落，请输入 ^p。 对于制表位，请输入 ^t。
              4. 选择所需的搜索选项。
              5. 选择“查找下一个”。
              6. 若要继续搜索，请选择“编辑”>“查找下一个”。
              请参阅
              替换文本

              替换文本
              1. 若要替换文档的某部分中的文本，请选择此部分。
                否则，将在整篇文档中搜索。
              2. 选择“编辑”>“替换”。
              3. 输入要查找的文本以及用来替换此文本的文本。
                对于段落，请输入 ^p。 对于制表位，请输入 ^t。
              4. 选择所需的替换选项。
              5. 选择“查找下一个”。
              6. 在“查找/替换”工具栏上，选择相应的按钮。

              移动、复制或删除文本
              1. 选择要移动、复制或删除的文本。
              2. 若要移动或删除文本，请选择“剪切”按钮。
                若要复制文本，请选择“复制”按钮。
              3. 若要粘贴信息，请将插入点移动到所需的位置，然后选择“粘贴”按钮。
              注意：
              • 若要撤消操作，请选择“编辑”>“撤消”。
              • 若要恢复操作，请选择“编辑”>“重复”。

                创建项目符号和编号列表
                1. 选择“项目符号”或“编号”按钮。
                2. 键入第一个项目。
                3. 按 Enter 键，输入下一个项目。根据需要，重复操作。
                4. 再次选择按钮可结束设置列表格式。

                注意：若要更改选定列表的编号样式，请选择“格式”>“段落”。在“格式”列表中，选择所需的样式。

                  设置默认字体
                  1. 选择“格式”>“字体”。
                  2. 选择所需的字体。
                  3. 选择“设置为默认值”。
                  4. 若要使所做的更改应用于以后创建的文档，请选择“是”。
                  注意：若要将默认字体和样式应用到选定的文本，请选择“应用默认值”。

                            插入符号
                            1. 将插入点置于要插入符号的位置。
                            2. 选择“工具”>“插入符号”。
                            3. 选择所需的符号，然后选择“插入”。
                            提示：
                            • 如果选择其他字体（或字体的子集），将显示其他符号集。

                            打印文档
                            1. 选择“文件”>“打印”。
                            2. 选择所需的打印选项。
                            提示：
                            • “端口”列出可用的打印机端口。
                            • 如果在“端口”列表中选择了“网络”，请在“网络路径”框中输入网络打印机的路径。

                            为文档设置密码
                            1. 选择“文件”>“密码”。
                            2. 键入并确认密码。
                            3. 若要保留密码，必须保存文档（“文件”>“保存”）。

                              关于文档转换

                                如果设备上安装了电子邮件程序，则当接收到的附件是由 Word 6.0 或更高版本创建的 Word 文档或模板时，WordPad 将在设备上转换文件。若要查看文档，只需在电子邮件中打开附件或在 WordPad 中打开文件。

                                  在转换过程中，某些格式属性可能会更改或丢失。若要避免永久丢失这些属性，请在查看文件后关闭它而不是将它保存在设备上。如果要保存文件，请使用其他文件名保存。

                                    有关在设备上执行转换的详细信息，请参阅下列主题。

                                        格式

                                          边框和底纹
                                          WordPad 中不显示边框和底纹，但是转换回 Word 文档后可恢复边框和底纹。在表格中使用时，底纹不能恢复，但简单的边框可以恢复。

                                            字符格式
                                            可保留并显示粗体、斜体、删除线、上标、下标以及隐藏文本。其它效果可能被更改或删除。

                                              颜色
                                              颜色可以保留并映射到设备中的可用颜色。

                                                字体和字号
                                                移动设备不支持的字体将被映射为 WordPad 中最接近的可用字体。

                                                  页面格式
                                                  转换到 WordPad 时，标题、页眉、页脚、列、页面设置信息以及样式表将被删除。WordPad 支持内置标题。

                                                    段落格式
                                                    WordPad 保留并显示制表位、对齐方式、项目符号、缩进、简单编号列表以及段落间距。


                                                      图片

                                                        转换过程中将删除图片。


                                                        目录

                                                          将保留目录文本及某些格式。但在右对齐制表位处设置的文本将换到下一行。


                                                          索引

                                                            将保留索引文本及某些格式。


                                                            表格

                                                              表格显示为由制表符分隔的文本。阅读包含换行文本、制表符或段落标记的单元格可能非常困难。由 WordPad 文件转换回 Word 文档时，可以正确显示和还原简单的表格。


                                                              OLE 对象

                                                                转换过程中将删除 OLE 对象。


                                                                修订标记、批注和注释

                                                                  修订标记可见但无法更改。转换过程中将删除批注和注释。
