      인터넷 연결 정보

        이동 통신을 통해 인터넷에 연결한 다음 웹을 탐색하고 전자 메일을 주고 받을 수 있습니다.

          장치에 기본 제공 모뎀이 장착되어 있지 않으면 모뎀 카드 또는 외부 모뎀을 설치해야 합니다. 설치에 대한 자세한 내용은 모뎀 제조업체의 지침을 참조하십시오. 모뎀을 설정한 후 ISP에 대한 연결을 만듭니다.

            ISP 계정 만들기

              참고 PPTP 계정은 해당 장치에서 지원되지 않습니다.

                ISP(인터넷 서비스 제공자)에 PPP(Point-to-Point Protocol) 계정을 설정하고 다음 정보를 지정합니다.

                • 액세스 전화 번호
                • 사용자 이름
                • 암호

                계정을 설정한 후 장치에 새 연결을 만듭니다. 이 연결을 만들 때 새 연결 만들기 마법사에서 제공하는 기본 TCP/IP 설정을 모두 사용할 수 있어야 합니다. 기본 설정을 사용하여 연결할 수 없는 경우 ISP에 특정 TCP/IP 정보와 주 DNS 및 보조 DNS 주소를 문의하십시오.

                  장치에 새 연결 만들기
                    1. 시작 > 설정 > 네트워크 및 전화 접속 연결을 선택합니다.
                    2. 새 연결 만들기 아이콘을 두 번 클릭합니다.
                    3. 새 연결 만들기 대화 상자에 해당 연결에 대한 이름을 입력합니다.
                    4. 전화 접속 연결을 선택합니다.
                    5. 다음 단추를 선택합니다.
                    6. 드롭다운 목록에서 사용할 모뎀을 선택합니다.
                    7. 연결할 때마다 전화 번호를 새로 입력하려면 구성을 선택합니다.
                      • 장치 속성 대화 상자에서 포트 설정 탭을 선택합니다.
                      • 연결할 때마다 전화 번호를 입력할 수 있도록 수동 전화 걸기를 선택합니다.
                      • 전화 옵션을 사용자 지정하려면 전화 걸기 옵션 탭을 선택하고 옵션 기본 설정을 지정합니다.
                      • 확인을 선택하여 구성을 적용합니다.
                    8. TCP/IP 설정을 선택합니다.
                      • 일반 탭에서 서버에서 할당한 IP 주소 사용을 선택했는지 확인합니다.
                      • 이름 서버 탭에서 서버에서 할당한 주소 사용을 선택했는지 확인합니다.
                      • 확인을 선택합니다.
                    9. 다음 단추를 선택합니다.
                    10. 전화 번호 대화 상자에 ISP의 액세스 전화 번호를 입력합니다.
                    11. 마침 단추를 선택합니다.

                    방금 만든 연결이 네트워크 연결 창에 아이콘으로 표시됩니다.

                    참고

                    • 기본 설정을 사용하여 연결할 수 없는 경우 ISP 또는 네트워크 관리자에게 문의하십시오.
                    • 외부 모뎀에 사용할 PC 모뎀 연결을 만들 경우 COM1의 Hayes 호환을 모뎀으로 선택합니다.
                    • 외부 모뎀을 사용하여 장치를 PC에 연결할 경우에는 모뎀 제조업체가 제공한 모뎀 케이블 또는 어댑터를 사용하십시오.

                        네트워크에 연결 정보

                          네트워크(이더넷)나 전화 접속 연결을 통해 네트워크에 직접 연결할 수 있습니다. 전자 메일에 액세스하고 네트워크 서버에서 사용할 수 있는 파일에 액세스하며 인터넷이나 인트라넷을 탐색할 수 있습니다. 전자 메일을 받으려면 네트워크에 연결한 다음 받은 편지함에서 POP3나 IMAP4 메일 서비스를 만듭니다. 자세한 내용은 받은 편지함 도움말을 참조하십시오.

                            참고 항목
                            네트워크(이더넷) 연결 만들기
                            전화 접속 네트워크 연결 만들기

                            네트워크(이더넷) 연결 만들기

                              1. 장치에 이더넷 카드를 삽입합니다.
                              2. 이더넷 드라이버 설정 대화 상자에서 DHCP를 사용하여 IP 주소 얻기를 선택합니다. 확인을 선택합니다.
                              3. 시작 > 설정 > 제어판을 선택합니다. 네트워크 및 전화 접속 연결 아이콘을 두 번 클릭합니다.
                              4. 어댑터의 연결 아이콘을 두 번 클릭하고 확인을 클릭하여 설정을 확인합니다.
                                예를 들어, DC21x4 Ethernet 어댑터를 사용한다면 DC211401 연결 아이콘을 두 번 클릭합니다.
                              5. 입력하라는 메시지가 표시되면 네트워크에 로그인할 때 사용하는 사용자 이름, 암호 및 도메인 이름을 입력합니다.

                                참고 항목
                                네트워크에 연결

                                전화 접속 네트워크 연결 만들기

                                1. RAS 계정을 만들려면 네트워크 관리자에게 문의하십시오. 계정이 설정되면 전화 접속 전화 번호, 암호, 컴퓨터 이름 등의 정보를 지정합니다.
                                2. 1단계의 정보를 사용하여 장치에 사용할 새 연결을 만듭니다. 새 연결 만들기 마법사에서 제공하는 기본 TCP/IP 설정을 사용할 수 있어야 합니다. 이러한 기본 설정을 사용하여 연결할 수 없는 경우에는 특정 TCP/IP 정보에 대해 네트워크 관리자에게 문의하십시오.
                                참고 항목
                                장치에 새 연결 만들기

                                장치에 새 연결 만들기

                                  이더넷 카드를 사용하지 않으면 원격으로 액세스할 컴퓨터나 네트워크마다 연결을 하나씩 만들어야 합니다.

                                  1. 시작 > 설정 > 네트워크 및 전화 접속 연결을 선택합니다.
                                  2. 새 연결 만들기 아이콘을 두 번 클릭합니다.
                                  3. 새 연결 만들기 대화 상자에 해당 연결에 대한 이름을 입력합니다.
                                  4. 전화 접속 연결을 선택하고 다음 단추를 클릭합니다.
                                  5. 사용할 모뎀을 선택합니다.
                                  6. 연결할 때마다 전화 번호를 새로 입력하려면
                                    • 수동으로 전화 걸기를 선택한 다음 확인을 클릭합니다.
                                    • TCP/IP 설정을 선택합니다.
                                    • 일반 탭에서 서버에서 할당한 IP 주소 사용을 선택했는지 확인합니다.
                                    • 이름 서버 탭에서 서버에서 할당한 주소 사용을 선택했는지 확인하고 확인을 선택합니다.
                                    • 다음 단추를 선택하고 전화 번호를 입력합니다.
                                    • 마침 단추를 선택합니다.
                                  방금 만든 연결이 원격 네트워킹 폴더에 아이콘으로 표시됩니다.
                                    참고
                                    • 이러한 기본 설정으로 연결할 수 없는 경우에는 특정 구성 또는 TCP/IP 정보에 대해 ISP나 네트워크 관리자에게 문의하십시오.
                                    • 연결이 연결 화면에 아이콘으로 표시됩니다.
                                    • 외부 모뎀에 사용할 PC 모뎀 연결을 만들 경우 COM1의 Hayes 호환을 모뎀으로 선택합니다. 외부 모뎀을 사용하여 장치를 PC에 연결할 경우에는 모뎀 제조업체가 제공한 모뎀 케이블 또는 어댑터를 사용하십시오.

                                    네트워크에 연결

                                    네트워크에 연결하면 전자 메일을 받거나 인터넷을 탐색하는 작업만 할 수 있습니다. 전자 메일을 받으려면 받은 편지함에서 POP3이나 IMAP4 메일 서비스를 만듭니다. 자세한 내용은 받은 편지함 도움말을 참조하십시오.

                                      1. 시작 > 설정 > 네트워크 및 전화 접속 연결을 선택합니다.
                                      2. 사용할 원격 연결 아이콘을 두 번 클릭합니다.

                                        해당 연결을 사용하여 처음 연결할 때 사용자 이름, 암호 및 도메인을 입력해야 합니다. 다음 연결부터 암호를 다시 입력하지 않으려면 암호 저장을 선택합니다.
                                      3. 연결을 선택합니다.
                                      4. 연결을 끊으려면 연결 끊기를 선택합니다.

                                      무선 네트워크에 연결
                                      1. 무선 카드를 연결합니다. 메시지가 표시되면 제조업체에서 제공한 드라이버 이름을 입력합니다.
                                      2. 일단 로드되면 사용 가능한 무선 네트워크에 연결하는 것을 도와 주는 대화 상자가 나타납니다.
                                      3. 연결할 네트워크 이름을 선택한 다음 연결을 클릭합니다.
                                      4. 검색된 설정을 확인한 다음 확인을 클릭합니다.
                                      5. 설정을 검토하거나 변경해야 하는 경우 시작 > 설정 > 네트워크 및 전화 접속 연결을 선택합니다.
                                      6. 사용할 원격 연결 아이콘을 두 번 클릭합니다.
                                      7. 사용하는 무선 카드 드라이버에서 지원하는 경우 무선 네트워크 탭을 사용하여 설정을 조정할 수 있습니다.
                                      8. 설정을 보거나 변경, 조정하려면 이 탭을 사용합니다.

                                        소프트웨어 압축 기능 해제

                                          장치의 전송 속도 변경

                                            중요 이 설정을 변경하면 PC와 통신하지 못할 수도 있습니다. 장치의 전송 속도는 PC의 전송 속도와 일치하도록 선택해야 합니다. PC의 전송 속도 변경에 대한 자세한 내용은 Windows Embedded Compact 서비스 도움말을 참조하십시오.

                                            1. 시작 > 설정 > 제어판을 선택합니다.
                                            2. PC 연결 아이콘을 두 번 클릭합니다.
                                            3. 변경...을 선택합니다.
                                            4. 원하는 전송 속도를 선택합니다.

                                              터미널 정보

                                                TTY 또는 VT-100 터미널 에뮬레이션을 필요로 하는 회사 서버 또는 온라인 서비스에 장치를 연결할 수 있습니다. 예를 들어, 게시판에서 파일을 보거나 다운로드할 수 있으며 전자 메일을 주고 받을 수 있습니다.

                                                  터미널 에뮬레이션을 필요로 하는 서비스와 통신할 때만 터미널을 사용하십시오.

                                                    새 터미널 세션 만들기
                                                    1. 시작 > 설정 > 제어판 > 터미널 서버 클라이언트 라이센스를 선택합니다.
                                                    2. 새 세션 만들기 아이콘을 두 번 클릭합니다.
                                                    3. 세션 이름 상자에 세션의 이름을 입력합니다.
